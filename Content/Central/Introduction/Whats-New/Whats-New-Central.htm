What's New?

Following are new features available in .

Task Boards

  • Create task boards to organize tasks
  • Existing tasks placed in default board
  • Move tasks from one board to another

Rich Text Editor for Tasks

  • Edit and format text in the task description
  • Add links to the task description

Host Mapped Domain Enhancements

  • New wizard
  • Default site avoids 404 pages at root domain
  • Microsoft Azure new SSL certificate binding service

Jira Integration

  • Connect Jira account with Central license
  • Associate Central task boards with Jira projects and boards
  • Push Central task boards to Jira so that Central tasks display in Jira as issues
