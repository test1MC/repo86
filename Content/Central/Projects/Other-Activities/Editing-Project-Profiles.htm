Editing Project Profiles

How to Edit a Project Profile

  2. Click the project icon.

  3. In the dialog, click Edit.

  4. From this dialog, you can use the tabs to modify any of the following:

    Settings Tab

    • Icon Color Click the square to the right, select a color, and click Confirm.
    • Description You can edit the project’s description.
    • Initials You can modify the initials that are shown in the project icon.
    • Name Click in the text field and change the name for the project.

    Teams Tab

    • You can click check boxes to determine which teams are associated with the project. All users on the selected teams will then be associated with the project.

    Users Tab

    • You can click check boxes to determine which users are associated with the project.
  5. Click Save.
