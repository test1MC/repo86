Security Page View

The Security page view lets you create groups of trusted domains to allow the publication of approved content. This feature might be necessary if you want to display one or more external sites within your output. For example, in one of your Flare topics maybe you have inserted a link to an external website within an IFrame. Without setting that domain as trusted, the site might fail to display due to certain security policies (see Security Headers and Trusted Domains)(see the Security Whitepaper PDF).
