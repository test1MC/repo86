Editing Host Mapped Domains

You can make changes to a host mapped domain that you've already added.

How to Edit a Host Mapped Domain

  1. On the left side of the interface, click Sites.
  2. Select the URLs tab.
  3. Click next to the domain you want to edit and select Edit.
  4. Make changes in the Host Mapped Domain wizard. For field details, see Adding Host Mapped Domains.For field details, see Adding Host Mapped Domains.
  5. Click Save.
  6. Click the x in the upper-right corner to close the wizard.
