Main Navigation Framework

If you are logged in with the author user type, you will see the entire user interface. If you are logged in as a subject matter expert (SME), you will only see the Reviews page and limited other parts of the interface.

Top Navigation

Shows your avatar and user name. If you do not have an avatar set, your initials will be shown. If you are a user administrator, you will see a crown on your avatar. Also, a colored circle indicates a user's status:

Online

Away (when the user is inactive for some time)

Offline

After clicking your name/avatar, you can choose the following:

  • User Settings Opens the User Settings dialog, which lets you edit details in your profile by making selections on the left (e.g., enter contact information, change password, assign teams and projects, set permissions and notifications). It's the same dialog that you can open for yourself and other users by clicking the user avatar or name in various places in the interface. See Setting Your Profile and Notifications.

  • License Settings Opens the License Settings dialog. Users with the appropriate administrative permissions can manage the Central license on the server. This includes the ability to change the license key label, set up Slack integration, and choose security settings. Also, some users are allowed to buy more user seats and purchase additional storage space. See Server Management and Purchasing.
  • Sign Out Lets you log out of the system.

Displays the label for your license key. All of the information you see throughout the interface is tied to this license.

If you have access to more than one Central license, you can click the down arrow to switch to another license without signing out.

Lets you see recent notifications. Alerts for new notifications are indicated by a red dot. See Setting Your Profile and Notifications.

After you click the icon, the alert will be removed from the icon.

Lets you open the Message Center, which is a conversation platform for all users on the license (see Message Center). Alerts for new messages are indicated by a green dot.

After you open your messages and then close the Message Center, the alert will be removed from the icon.

After clicking this, you can choose the following:

  • Documentation Opens the online Help in a separate browser tab.
  • Submit Feedback Opens a web page in a separate browser tab, where you can submit bugs and provide other feedback.
  • Privacy Policy Opens MadCap Software’s privacy policy in a separate browser tab.
  • Terms of Use Opens MadCap Software’s terms and conditions in a separate browser tab.
  • About Opens a popup showing basic information about MadCap Central. You can also click the Central icon in the lower-left corner of the interface to see this.

Left Navigation

Opens the Home page dashboard, where you can add widgets pointing to various information in the system. See Dashboards and Widgets.

Opens the Projects page, where you can manage uploaded Flare projects and builds. See Projects.

Opens the Analytics page, where you can view analytics on published Flare HTML5 output. This includes search phrases used, search phrases with no results, topics viewed, context-sensitive Help calls, and demographic statistics (browsers and operating systems). See Analytics.

Opens the Sites page, where you can manage and view sites on the license. See Sites.

Opens the Reviews page, where you can view, edit, and manage Flare review files. See Reviews.

Opens the Task Boards page, where you can create, edit, and view tasks. See Tasks.

Opens the Teams page, where you can create and manage teams. See Teams.

Opens the Users page, where you can manage and view users on the license. See Users.

Opens a dialog, where you can request a translation quote from MadTranslations for your projects and other files. See Requesting a Translation Quote.
