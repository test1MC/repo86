Grids

Multi-Select

By selecting check boxes to the left of the rows, you can take action on multiple items at once (e.g., delete them, change a status). If you click the check box at the top, it will select all check boxes under it.

How to Filter Columns in a Grid

  1. Open a page with a grid—such as Projects, Tasks (Grid, Archive, or Backlog), Teams, or Users.

  2. In the grid, click in the column you want to filter.

    If you do not see in a column, you cannot filter that column.

  3. Complete the options in the popup and click Filter.
  4. To return the grid to displaying all items, click , and in the popup select Clear.

How to Show and Hide Columns in a Grid

  1. In Central, open a page with a grid—such as Projects, Tasks (Grid, Archive, or Backlog), Teams, or Users.

  2. In the toolbar, click .

    If you do not see this option in the local toolbar, you cannot show or hide columns for the current page.

  3. Select the columns you want to display, and deselect the ones that you don't.

  4. Click Save. The columns on the page will be updated.

In all grids in Central, horizontal scroll bars display if you have enabled more columns than can be shown within the window. This allows for better access to all columns in the grid.

The Builds page has a lot of column options that can be added by clicking in the toolbar. If you decide to add all of them, notice that the window is not wide enough to show them all at once. However, you can use the scroll bar at the bottom.

When you scroll to the right, you can see the rest of the columns that normally would have been hidden from view.

Context Menu

In various places throughout the interface, you might notice some context (three-dot) menus. After clicking these dots, you will see a drop-down menu, letting you select options related to that part of the interface. In some cases, right-clicking will also display a context menu; however, sometimes it shows only options related to your browser.
