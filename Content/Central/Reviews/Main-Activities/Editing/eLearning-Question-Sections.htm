eLearning Question Sections

If an author sends a topic or snippet for review that includes an eLearning question section (i.e., question, answers, and sometimes feedback and a Submit button), it displays in the Review Editor and can be edited. For more details about learning & development (i.e., eLearning), see the Flare online Help.

How to Edit an eLearning Question Section

  1. On the Reviews page, open a topic or snippet that contains a question section.
    • Replace or delete text for the question, answers, or feedback.
    • Select an answer's circle or square so that it is marked as correct.
    • Press ENTER at the end of any answer to add a new answer.
    • If you want to add other content below the question section, press the down arrow on your keyboard until you see the horizontal cursor at the very end. Then, press ENTER to create a new empty paragraph.
