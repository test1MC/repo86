Images

You can insert and move images in Central's Review Editor.

In order to use this feature in Central, the file to be reviewed must have been sent from the latest version of Flare.

Inserting New Images

After opening a file for review, you can insert new images in it.

How to Insert an Image

  1. In the local toolbar of the Review Editor, click .

  2. In the dialog, navigate to the image on your computer, select it, and click Open.

File Types Supported

  • PNG

  • GIF

  • JPG

  • BMP

  • SVG

  • TIFF

File Size Limit

15 MB

Moving Images

You can move images around in the Review Editor by clicking the image, dragging it, then dropping it at the cursor location.

Accepting Images in Flare

After reviewed topics and snippets are submitted, they are added to the original author's review inbox in Flare.

If the author accepts the file back into the Flare project, any images that were added in Central will be stored in the Resources > Images folder in Flare by default. If the Images folder doesn't yet exist, Flare will create it for you.

If there is a conflict (e.g., you already have an image in the project with the same name), a wizard opens so that the author can resolve the conflicts. The author can select a different location in Flare to store the image and/or give it a different name.

At this time, you cannot resize images in the Review Editor in Central. The author can resize the image once it is accepted back into the Flare project.
