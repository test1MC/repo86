Toolbar

You can use the options in the toolbar at the top of the Review Editor to accomplish different tasks.

Reverses the most recent action.

Repeats the most recent action that you reversed.

Removes the selected annotation from the document.

Applies a paragraph tag to the content.

Applies a heading tag (H1 through H6) to the content.

Applies a bulleted list tag to the content. See Lists.

Applies a numbered list tag to the content. See Lists.

Applies definition list tags to the content. See Lists.

Outdents the list item(s). This option pertains only to lists, not other kinds of content. See Lists.

Indents the list item(s). This option pertains only to lists, not other kinds of content. See Lists.

Opens a drop-down that lets you select the number of columns and rows that you want to add as you insert a new table. See Tables.
