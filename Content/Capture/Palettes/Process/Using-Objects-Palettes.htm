Using Objects From Palettes

After objects are added to a palette, you can use them in images whenever necessary.

How to Use an Object From a Palette

  1. Open an image.
  2. Open the palette containing the object that you want to use.
  3. Double-click the object in the palette, or click and drag it to the image.
  4. You can then move, resize, or double-click the object to make property changes to it.
