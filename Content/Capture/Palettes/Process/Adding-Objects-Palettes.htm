Adding Objects to Palettes

After you create or open a palette, you can add objects to it.

How to Add an Object to a Palette

  1. Open a custom palette that you have created, or a linked palette (not a factory palette).
  2. Open an image and right-click the object you want to add to the palette.
  3. In the context menu, select Add to Palette. The object is added to the Palettes window pane.
  4. (Optional) You can rearrange the order of the objects in the Palettes window pane by clicking and dragging the objects up or down.

Objects can be added only to custom palettes that you have created or to linked palettes. They cannot be added to read-only factory palettes.

After you add an object to a palette, you can use that object in other images. See Using Objects from Palettes.
