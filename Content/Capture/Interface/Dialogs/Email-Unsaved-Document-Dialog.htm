Email Unsaved Document Dialog

This dialog opens if you have an image that you have not yet saved and you try to send it to an email recipient. The dialog lets you select a file type and enter a name for the image file. See Sending Images to an Email Recipient.

How to Open This Dialog

Select File > Send To > E-mail Recipient.

Buttons and Sections

Select file format

Select the file type for the image.

Optional file name

Enter a file name or use the default name provided.
