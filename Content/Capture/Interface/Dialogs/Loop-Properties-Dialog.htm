Loop Properties Dialog

You can use this dialog to accomplish several editing tasks for loop shapes, including the following.

  • Blur Effect (Image Effects Tab) Read more… See Adding Blur Effects.
  • Blur-Inside Effect (Image Effects Tab) Read more… See Adding Blur-Inside Effects.
  • Color (Appearance Tab) Read more… See Setting the Color for an Object.
  • Condition Tags (Conditions Tab) Read more… See Applying Conditions.
  • Coordinates (Loop Tab) Read more… You can specify the coordinates of a loop to change how the loop looks. See Setting the Coordinates for a Loop.
  • Font Properties (Font Tab) Read more… See Setting Font Properties for Text.
  • Gray Scale Effect (Image Effects Tab) Read more… See Adding Gray Scale Effects.
  • Language (Language Tab) Read more… See Selecting a Language.
  • Padding (Appearance Tab) Read more… See Adding Padding to Objects.
  • Shading Effect (Image Effects Tab) Read more… See Adding Shading Effects.
  • Shadow Effect (Shadow Tab) Read more… See Adding Shadow Effects.
  • Text (Rich Text Tab) Read more… See Adding Text.
  • Thickness (Loop Tab) Read more… You can specify the thickness of a loop that you add to an image a frame. See Setting the Thickness for a Loop.
