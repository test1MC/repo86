Creating Variables

You can create new variables in images, although a more common task is to link to existing variables from a MadCap Flare project. See Linking to Flare Projects.

You can create variables in images using the File Properties dialog or the Profiles Editor. Use the Capture Editor if you want to create variables for a single image only. Use the Profiles Editor if you want to create variables for a profile, which can be used when capturing future images. See Profiles.

How to Create Variables Using the File Properties Dialog

  1. Open the image and double-click it.
  2. In the properties dialog, select the Variables tab.

  3. To enter a new name, definition, or comment for a new variable, click in the appropriate cell. Then press F2 on your keyboard and type the name, definition, or comment. Press ENTER after you finish typing. As soon as you begin typing in one of the cells, a new blank row is added in the tab (to be used for the next variable that you create).

  4. Click OK.

How to Create Variables Using the Profiles Editor

  2. Select the Variables tab.
  3. To enter a new name, definition, or comment for a new variable, click in the appropriate cell. Then press F2 on your keyboard and type the name, definition, or comment. Press ENTER after you finish typing. As soon as you begin typing in one of the cells, a new blank row is added in the tab (to be used for the next variable that you create).
  4. Click OK.

Adding a comment is optional and for your internal use only.

After you create a variable, you can insert it into an object that has been added to the image. See Inserting Variables Into Objects.
