Pinning Fonts

How to Pin Fonts From the Object Properties Dialog

  1. In the Capture Editor, double-click the object (not the image).
  2. In the properties dialog, select the Rich Text tab or the Font tab.

  3. In the [Font Family] field, select the font you want to pin. A pin appears to the right of the font. You can pin fonts from any font group.

  4. To pin the font to the font list, click . The pin changes to to show that the font has been pinned to the font list. Pinned fonts are also added to the Pinned Fonts group at the top of the font list.

  5. To unpin the font, click the pin again and it will be moved back to All Fonts group at the bottom of the list.

How to Pin Fonts From the Popup Text Entry Window

  1. In the Capture Editor, click the object (not the image).
  2. Start typing content or press F2.
  3. In the toolbar at the top of the popup, click the down arrow in the Font Family field and find the font you want to pin. A pin appears to the right of the font. You can pin fonts from any font group.

  4. To pin the font to the font list, click . The pin changes to to show that the font has been pinned to the font list. Pinned fonts are also added to the Pinned Fonts group at the top of the font list.

  5. To unpin the font, click the pin again and it will be moved back to All Fonts group at the bottom of the list.

After you complete these steps, you can select fonts from the Pinned Fonts group when applying them to content. See Adding Text.
