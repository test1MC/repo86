Capturing a Window

Use this method to capture a fixed area of a window (e.g., menu bar, toolbar, editor, individual button, entire window), depending on where you move your mouse and click. As you move the mouse in an application, a red border surrounds each separate region.

How to Capture a Window

  1. Open the window or a user interface (UI) area that you want to capture. When you initiate a window capture, the Capture application will be minimized, allowing you to "grab" an area directly behind it.
  3. Do one of the following:
    • Ribbon Select File > Capture > Capture Window or Edit > Capture > Capture Window.
    • Keyboard Shortcut Press CTRL+SHIFT+W.

    4. Capture minimizes and a red border surrounds each window or UI element as you move your cursor over it.

  4. Click the window or UI element that you want to capture. The image is captured and opens in the Capture Editor.

    You can also perform a UI element object capture. This lets you insert one image as an object on top of another image. See Inserting Captured UI Elements as Objects.
