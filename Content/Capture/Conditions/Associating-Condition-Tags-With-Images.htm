Associating Conditions With Images

You can associate condition tags with images using the File Properties dialog or the Profiles Editor. Use the File Properties dialog if you want to associate condition tags with a single image only. Use the Profiles Editor if you want to associate condition tags with a profile, which can be used when capturing future images. See Profiles.
