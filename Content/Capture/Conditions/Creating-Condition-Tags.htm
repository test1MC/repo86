Creating Conditions

You can create condition tags using the File Properties dialog or the Profiles Editor. Use the File Properties dialog if you want to create condition tags for a single image only. Use the Profiles Editor if you want to create condition tags for a profile, which can be used when capturing future images. See Profiles.

How to Create Conditions Using the File Properties Dialog

  1. Double-click the image.
  2. In the File Properties dialog, click the Condition Tags tab.
  3. Click in an empty Name cell and type a name for the tag.
  4. Click the Background drop-down arrow and select Pick Color.
  5. Select a new color and click OK.
  6. (Optional) Click in the Comment cell, type an internal comment, and press ENTER.

How to Create Conditions Using the Profiles Editor

  2. In the Profiles Editor, select the Condition Tags tab.
  3. Click in an empty Name cell and type a name for the tag.
  4. Click the Background drop-down arrow and select Pick Color.
  5. Select a new color and click OK.
  6. (Optional) Click in the Comment cell, type an internal comment, and press ENTER.

After you create condition tags, you can apply them to objects in the image. See Applying Conditions.
