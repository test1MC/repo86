Editing Text

After you add text to an object, you can edit it as necessary.

How to Edit Text

  1. Open the image and click the object containing text.
  2. Double-click the object, or right-click it and select Edit Text. Either a dialog or popup opens. If you opened a dialog, select the Rich Text tab.
  3. Edit the text as necessary. You can use the buttons and drop-downs at the top to apply formatting to any part of the text, or to insert a variable.
  4. When you are finished, click OK in the dialog, or click outside of the popup window.
