Setting Font Properties for Text

How to Set Font Properties

  1. Open an image and double-click the object containing text.
  2. In the properties dialog, select the Rich Text or Font tab.
  3. If using the Rich Text tab, select the text to which you want to apply the formatting changes.

  4. Make formatting changes using the appropriate options or fields.

You can pin fonts from the [Font Family] field. This is a way to limit the list of fonts to those that you use the most and hide from view those that you do not use. See Pinning Fonts.
