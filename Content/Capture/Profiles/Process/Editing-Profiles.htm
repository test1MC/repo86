Editing Profiles

You can open and edit a profile, specifying options such as the destination, effects, and appearance.

How to Edit a Profile

You can override the profile settings for an individual file if you want. To do this, simply double-click on the image and complete these same tabs and fields in the File Properties dialog.

After you edit a profile, you can select it so that it becomes the "current" (or active) profile. Then, when you capture an image, the settings in the profile are applied to the captured image. See Selecting a Profile for Capturing and Capturing an Image.
