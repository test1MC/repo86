Selecting a Profile for Capturing

In order to use a profile, you must first select it so that it becomes the active profile.

How to Select a Profile

  1. Select Edit > Profile.
  2. From the drop-down, select a profile. The name of that profile is displayed in the Profile field.

After you select a profile, you can capture an image. The settings in the profile are applied to the captured image. See Capturing an Image.
