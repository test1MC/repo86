Opening Profiles

You can open profiles from inside the interface or by dragging the files from a Windows folder.

How to Open a Profile From Within the Interface

  1. Select View > Profiles.
  2. In the upper-left of the local toolbar, click the down arrow and select the profile that you want to open.

How to Open a Profile by Dragging It From Windows

  1. Open Windows and navigate to the folder containing your profile (CAPRO file). By default, profiles are stored in your Documents/My Profiles folder.

  2. Launch .

  3. Drag the CAPRO file from Windows to the application window and drop it on the title bar in .

After you open a profile, you can edit the settings within it. See Editing Profiles.
