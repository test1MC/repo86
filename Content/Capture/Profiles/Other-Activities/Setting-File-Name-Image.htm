Setting the File Name for an Image

You can provide a file name for a captured image when you save the file. However, you also have the option of providing a default file name in a profile. That file name will then be applied to all images captured using the profile. The first image will be saved with the name you provide. Subsequent images will also be saved with that name, along with a unique number (e.g., filename1, filename2, filename3).

How to Set the File Name for an Image

  2. In the Profiles Editor, select the General tab.
  3. In the Output File Name field, type a name to be applied to all images captured with this profile. If you want the autonumbers to be placed somewhere other than at the end of the file name, add {n} to the appropriate location in the file name. (e.g., if you enter "{n}filename," the images will be saved as 1filename, 2filename, 3filename, and so on).
  5. (Optional) Capture an image and then select File > Save. Notice that the appropriate name is already provided in the "File name" field of the Save As dialog.
