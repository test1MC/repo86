Adding Padding to Objects

How to Add Padding to an Object

  1. Open an image and double-click the object (not the image).
  2. In the properties dialog, select the Appearance tab.

  3. In the Padding section, enter numbers in the Left, Right, Top, and/or Bottom fields to set the width of the padding (in pixels).

    You can also use the Copy Down button to enter the padding values more quickly. After you enter a number in the first padding field (Left), you can click this button to automatically enter the same number in the rest of the fields. This field is available only for certain objects.

  4. The padding is added to the object, changing the space between the edge of the object and any text within it.

Padding can be added only to certain objects.
