Auto-Sizing Capture Image Objects to Fit Text

If you have inserted MadCap Capture images in Flare projects—and those images contain objects with text—you can auto-size the objects automatically when the output is generated. This can be done by selecting an option in the Advanced tab of the Target Editor in MadCap Flare. The original image file and its associated properties (.props) file remain unchanged. Only the output image is affected.

How to Auto-Size Capture Image Objects to Fit Text

  1. In a Flare project, open a target.

  2. On the Advanced tab of the Target Editor, select Auto-size Capture objects to fit text.
