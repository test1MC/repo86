Editing Image Objects

After you add an image as an object, you can edit the object in various ways.

How to Modify the Position of an Image Object

The easiest way to modify the position of an image object is to click on the object and drag it. Alternatively, you can use the following steps.

  1. Open the image, and double-click the image object.
  2. In the Image Properties dialog, select the Image tab.
  3. Change the numbers in the X and Y fields.
    • X Sets the position of the object on the "X" axis (left and right) on the image.
    • Y Sets the position of the object on the "Y" axis (up and down) on the image.

How to Resize an Image Object by Dragging

  1. Open the image and select the image object.

  2. Click any point (i.e., small circle) around the edge of the object and drag it to adjust the width and/or height.

How to Resize an Image Object by Using the Image Properties Dialog

  1. Open the image, and double-click the image object.
  2. In the Image Properties dialog, select the Image tab.
  3. Change the numbers in the Width and Height fields.
    • Width Sets the width of the image object (in pixels).
    • Height Sets the height of the image object (in pixels).

How to Modify the Transparency of an Image Object

  1. Open the image, and double-click the image object.
  2. In the Image Properties dialog, select the Image tab.
  3. In the Transparency field, set the amount of transparency applied to the image object. The higher the number, the more transparent the image object will be. The lower the number, the more solid the image object will be.

How to Apply a Torn Edge Effect to an Image Object

  1. Open the image, and double-click the image object.
  2. In the Image Properties dialog, select the Edge Effects tab.
  3. Click the down arrow next to the Edge Effect field and select Torn.

  4. Complete the rest of the fields on the tab.

    • Wave Length Set the length of the "waves" for a torn edge effect (in pixels). This changes the width of the torn areas in the effect.
    • Wave Height Set the height of the "waves" for a torn edge effect (in pixels). This changes the depth of the torn areas in the effect.
    • Edges Click in the check boxes to select the specific edges (top, bottom, left, right) where you'd like the effect to be applied in the image. If a check mark is displayed, the effect will be applied to that edge.

How to Remove an Image Object

  1. Open the image, and click once on the image object.
  2. Press Delete.
