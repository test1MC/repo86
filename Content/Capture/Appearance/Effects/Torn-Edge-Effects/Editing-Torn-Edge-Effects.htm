Editing Torn Edge Effects

After you add a torn edge effect to an image, you can edit the effect in the various ways.

How to Modify the Wave Length and Height

  1. Open the image and double-click it.
  2. In the File Properties dialog, select the Edge Effects tab.
  3. Change the numbers in the Wave Length and/or Wave Height fields. The higher the number, the wider or deeper the wave in the torn edge.

How to Modify Which Edges are Torn

  1. Open the image and double-click it.
  2. In the File Properties dialog, select the Edge Effects tab.
  3. In the Edges section, click in the check boxes to add or remove check marks. This determines the specific edges (top, bottom, left, right) where you'd like the effect to be applied in the image. If a check mark is displayed, the effect will be applied to that edge.

How to Remove the Torn Edge Effect

  1. Open the image and double-click it.
  2. In the File Properties dialog, select the Edge Effects tab.
  3. Click the down arrow next to the Edge Effect field and select (default).
