Applying Gray Scale to Images

How to Apply Gray Scale Using the File Properties Dialog

  1. Capture or open the image.
  2. In the Capture Editor, double-click the image.
  3. In the File Properties dialog, select the Format tab.
  4. From the Medium drop-down, select the medium whose gray scale settings want to want to edit. If necessary, select Enable Format to enable the medium. See Selecting a Medium for an Image.
  5. Select Gray Scale so that a check mark displays in the box.

How to Apply Gray Scale Using the Profiles Editor

  2. In the Profiles Editor, select the Format tab.
  3. From the Medium drop-down, select the medium whose gray scale settings want to want to edit. If necessary, select Enable Format to enable the medium. See Selecting a Medium for an Image.
  4. Select Gray Scale so that a check mark displays in the box.

More About the Format Tab

You can also add a gray scale effect around an object in an image. Doing this will display the image in gray, except for the area inside the object, which remains displayed in color. For more information see Adding Gray Scale Effects.
