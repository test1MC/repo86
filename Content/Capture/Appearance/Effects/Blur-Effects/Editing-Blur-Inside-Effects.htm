Editing Blur-Inside Effects

After you add a blur-inside effect to an image, you can edit the effect or remove it.

How to Modify the Amount of Blur

  1. Open the image.
  2. Double-click the image (not the shape).
  3. In the File Properties dialog, select the Image Effects tab.
  4. Change the number in the Blur Factor field. The higher the number, the more the image is blurred.

How to Remove the Blur-Inside Effect

  1. Open the image.
  2. Double-click the shape (not the image).
  3. In the properties dialog for the object, select the Image Effects tab.
  4. Click in the Enable Blur-Inside Effect field to remove the check mark.

If you want to edit the blur effect around a shape, instead of inside it, see Editing Blur Effects.
