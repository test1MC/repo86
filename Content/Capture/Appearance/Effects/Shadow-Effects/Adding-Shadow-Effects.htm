Adding Shadow Effects

You can add shadow effects to images using the File Properties dialog or the Profiles Editor. Use the File Properties dialog if you want to add a shadow effect to a single image only. Use the Profiles Editor if you want to add a shadow effect to a profile, which can be used when capturing future images. See About Profiles.

How to Add a Shadow Effect to an Object

  1. Open an image that has an object added to it.
  2. Select the object.
  3. Select Object > Shadow. A shadow is added to the object.
  4. To make adjustments to the shadow, double-click the object.
  5. In the properties dialog, select the Shadow tab.

  6. (Optional) You can modify the position, color, and transparency of the shadow.

    To Modify the Position of the Shadow

    1. In the Left/Right field, enter the number of pixels that the shadow will be extended to the right or left of the object. Enter a positive number (e.g., 7) to extend the shadow to the right. Enter a negative number (e.g., -7) to extend the shadow to the left.
    2. In the Up/Down field, enter the number of pixels that the shadow will be extended below or above the object. Enter a positive number (e.g., 7) to extend the shadow below the object. Enter a negative number (e.g., -7) to extend the shadow above the object.

    To Modify the Color of the Shadow

    In the Color field, click the down arrow to select a color for the shadow. To see advanced color options, select More colors.

    To Modify the Transparency of the Shadow

    In the Transparency field, enter the percentage of transparency applied to the shadow.

How to Add a Shadow Effect to an Image Using the File Properties Dialog

  1. Open an image and double-click it.
  2. In the File Properties dialog, select the Shadow tab.
  3. Click Enable Shadow.

  4. (Optional) You can modify the position, color, and transparency of the shadow.

    To Modify the Position of the Shadow

    1. In the Left/Right field, enter the number of pixels that the shadow will be extended to the right or left of the image. Enter a positive number (e.g., 7) to extend the shadow to the right. Enter a negative number (e.g., -7) to extend the shadow to the left.
    2. In the Up/Down field, enter the number of pixels that the shadow will be extended below or above the image. Enter a positive number (e.g., 7) to extend the shadow below the object. Enter a negative number (e.g., -7) to extend the shadow above the object.

    To Modify the Color of the Shadow

    In the Color field, click the down arrow to select a color for the shadow. To see advanced color options, select More colors.

    To Modify the Transparency of the Shadow

    In the Transparency field, enter the percentage of transparency applied to the shadow.
