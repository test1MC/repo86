Using Color Fill

The color fill tool lets you fill an area with a color of your choosing. You can use this tool on objects or on the canvas.

How to Fill an Object or Area

The color fill tool lets you quickly fill areas of an image with a color.

  1. Open the image.
  2. From the Edit ribbon or local toolbar, click to select a color.
  4. (Optional) Zoom in on the image to select fill very specific sections, such as a line or a border. See Using the Zoom/Scale Feature.
  5. Click the object or canvas area you want to fill.

How to Fill Using the Selection Rectangle

You can use the selection rectangle to set a boundary for the fill area. This lets you limit the area where you can fill to the area within the rectangle's borders.

  1. Open the image.
  2. From the Edit ribbon or local toolbar, click .
  3. Click on the part of the canvas you want to fill and drag the mouse until the rectangle covers the section of the canvas you want to fill.
  4. From the Edit ribbon or local toolbar, click to select a color.
  6. (Optional) Zoom in on the image to fill very specific sections, such as a line or a border. See Using the Zoom/Scale Feature.
  7. Click on the object or canvas area you want to fill.

 

You have a picture with a large background image. The background image is detailed, and you need a solid background area for text to stand out. You can use the selection rectangle to create a uniform, square shape to use for your text.

Then, use the color fill tool to fill the area with the desired color. Depending on the detail of your image, you may need to set the tolerance to a higher level.

How to Fill Using the Magic Wand

You can use the magic wand to select a specific area. This lets you limit the area where you can fill to the magic wand's selection.

  1. Open the image.
  2. From the Edit ribbon or local toolbar, click .

  3. Click on the part of the canvas you want to fill. The magic wand adds a semi-transparent overlay onto the surface of the selection.

  4. From the Edit ribbon or local toolbar, click to select a color.
  6. (Optional) Zoom in on the image to fill very specific sections, such as a line or a border. See Using the Zoom/Scale Feature.
  7. Click on the object or canvas area you want to fill.

 

Your background image includes text. Because the text is set smoothly onto the image, pixels at the edges of the letters go from lighter to darker. This makes using the color fill tool by itself more difficult; you'd have to click on each of those pixels to color each letter entirely.

Instead, select the part of the image you want to correct using the magic wand tool. Use the tolerance drop-down to select a higher level of tolerance for the wand. This will allow the wand to ignore slight color variations and select a larger area.

Once you're satisfied with the selection, use the color fill tool to replace the selected areas with a solid color.
