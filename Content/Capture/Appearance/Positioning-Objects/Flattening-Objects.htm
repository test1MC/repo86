Flattening Objects

The flatten feature lets you merge selected objects with the background image, combining them into a single image. This is useful when bringing in images from another tool, because you can make them a permanent part of your background image. This makes it easier to add and manipulate additional objects.

Example

How to Flatten Objects

  1. Open an image and select an object in it.

    • Ribbon Select Object > .
    • Layout Toolbar Select .
    • Right-Click Select Flatten.

    The object(s) is merged into the image.

Any objects outside of the true canvas area will be cropped out when the image is flattened. If an object is partially outside of the true canvas area, it will be cropped at the true canvas border.

Once you flatten an object into the image, the object layer is removed. You will not be able to edit the object properties once it has been flattened.
