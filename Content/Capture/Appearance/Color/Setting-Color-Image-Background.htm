Setting the Color for an Image Background

You can set the color for an image background using the File Properties dialog or the Profiles Editor. Use the File Properties dialog if you want to set the background color for a single image only. Use the Profiles Editor if you want to set the image background color for a profile, which can be used when capturing future images. See About Profiles.

Example

 

If you have added an object to an image and then dragged the object outside the current boundaries of the image, padding is automatically added to compensate for the space needed. When you do this, the image background is exposed in the default transparent color (gray and white checks). Therefore, you may want to set the background color (e.g., white) to match the canvas where you plan to insert the finished image.

How to Set the Background Color Using the File Properties Dialog

  1. With the image open in the Capture Editor, double-click the image.
  2. In the File Properties dialog, select the Appearance tab.

  3. Use the Background section to set the color properties.

How to Set the Background Color Using the Profiles Editor

  2. In the Profiles Editor, select the Appearance tab.

  3. Use the Background section to set the color properties.
