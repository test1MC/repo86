Specifying the Page Setup

When you want to print an image file from Capture, you can first specify the page settings (paper, orientation, and margins).

How to Specify the Page Setup

  1. Capture or open the image.
  2. Select File > Page Setup.
  3. In the dialog, complete the options in the dialog as necessary.
  4. Click OK.
  5. Now you can print the image file using those settings. See Printing Images.
