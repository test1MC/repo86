Setting Image Properties Using Mediums

When applying image properties, you must first select the medium you want to use: web (default), print, or custom. The image properties you select are saved for each medium type and used in Flare outputs associated with that medium.

How to Set Image Properties Using Mediums

  1. Open the image, and double-click it.
  2. In the File Properties dialog, select the Format tab.

  3. From the Medium drop-down, select the medium you want to want to edit (i.e., print, web, custom). If you selected a custom medium, the Medium Source field displays the location of the stylesheet file associated with that medium.

  4. (Optional) If the settings for the medium are disabled, select Enable Format to enable changes.

  5. Make changes to the properties settings. See Setting the File Format for an Image, Setting the JPEG Quality for an Image, Setting the Color Depth for an Image, Setting the Print DPI for an Image, and Resizing Images.

  6. In the Preview section, view your settings. You can click Enable Preview to toggle the Preview section on or off.
