Creating New Images

Rather than capturing an image, you can create a new image using settings from a profile (see Profiles). Aside from the settings of that profile, the new image will start out blank.

How to Create a New Image

  1. Select File > New > New or press CTRL+N. The Create New Image dialog opens, displaying all of your profiles.
  2. Select a profile.
  3. The new blank image opens in the Capture Editor.
  5. Navigate to a location to store the image (if necessary), provide a name for the image, and click Save.

After an image file is created, you can use all of the standard tools in Capture to change it (e.g., adding objects, choosing a background color, changing the canvas size). See Editing Images.
