Running Quality Assurance Reports

How to Run Quality Assurance Reports

  1. Select File > Project QA. The Quality Assurance dialog opens.

  2. Select the check boxes for the quality assurance reports you want to run.

    • This report is unavailable if an MT provider is not connected to the project.

  4. The QA report(s) open in a new tab.

  5. (Optional) You can print or save your report. In the local toolbar, click to print the report. Click to save the report as an HTML file.
