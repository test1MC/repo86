Mapping Fonts

You can create, edit, and delete font maps from the Manage Font Maps dialog.

How to Create a Font Map

  1. Select Resources > Manage Font Mapping.

    The Manage Font Maps dialog opens.

  2. Click . The Create New Font Map dialog opens.
  3. In the field, enter a name for the font map.
  5. In the Font Map Files dialog, click OK to close the dialog, or edit the new font map to assign font mappings.

How to Edit a Font Map

  1. Select Resources > Manage Font Mapping.

    The Manage Font Maps dialog opens.

  2. Select the font map you want to edit.
  3. Click . The Modifying Font Map dialog opens.

  4. From the Language drop-down, select the target language for which you want to map fonts.

    You can only select target languages currently associated with the project.

  5. In the File grid, select the files that you want to scan for fonts.

  6. Click Discover Fonts. Lingo scans the file(s) you selected. Located fonts appear in the Original column of the grid at the bottom of the dialog.

  7. From the Replace drop-downs next to each Original font, do one of the following:
    • Choose a New Font Select a named font if you want to use a different font for the selected target language.
    • Keep the Original Font Select (none) if you want to keep the original font.
  8. (Optional) Repeat steps 4-7 for each additional target language.
  10. (Optional) If you want to use this font map in your project, select the check box in the In Use column. You can use one font map per project.

How to Delete a Font Map

  1. Select Resources > Manage Font Mapping.

    The Manage Font Maps dialog opens.

  2. Select the font map you want to delete.
  3. Click .

  4. When asked if you are sure you want to delete the font map, click OK.

    Deleting a font map is permanent. Deleted font maps are removed from all Lingo projects.

  5. In the Font Map Files dialog, click OK.
