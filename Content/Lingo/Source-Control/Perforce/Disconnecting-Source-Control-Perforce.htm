Disconnecting From Source Control—Perforce Helix Core

How to Disconnect From Perforce Helix Core

  1. Check out the files you will need to work on while disconnected. See Checking Out Source Control Files—Perforce Helix Core.

    • Ribbon Select Source Control > Disconnect.

How to Reconnect to Perforce Helix Core

    • Ribbon Select Source Control > Reconnect.

  2. Submit the files you previously checked out. See Submitting Source Control Files—Perforce Helix Core.
