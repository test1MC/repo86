Binding a Project to Microsoft Visual Studio Team Services

includes Microsoft Team Foundation Server support for Microsoft Visual Studio Team Services (VSTS). If you manage your source control projects with Microsoft's online version of Visual Studio, you can bind your projects to and perform check-ins and check-outs of files.

Use the following steps if you have already created a project and want to bind ("connect") it to the Microsoft VSTS repository using Microsoft Team Foundation Server.

After you bind a Microsoft VSTS project using Microsoft Team Foundation Server in , you can use the interface in the same way as you would with any other project bound to Microsoft Team Foundation Server.
