Disconnecting From Source Control—Team Foundation Server

How to Disconnect From TFS

When you want to work offline, you can disconnect from source control.

  1. Check out the files you will need to work on while disconnected. See Checking Out Source Control Files—Team Foundation Server.

    • Ribbon Select Source Control > Disconnect.

    • Right-Click If you have the File List open, right-click on any file and select Source Control > Project > Disconnect.
    • Local Toolbar In the local toolbar of the File List, click , then select Project > Disconnect.
  3. A confirmation dialog appears. Click Yes. You will be disconnected from source control.

How to Reconnect to TFS

When you are finished working offline, you can reconnect to source control.

    • Ribbon Select Source Control > Reconnect.

    • Right-Click If you have the File List open, right-click on any file and select Source Control > Project > Reconnect.
    • Local Toolbar In the local toolbar of the File List, click , then select Project > Reconnect.

  2. Check in the files you previously checked out. See Checking In Source Control Files—Team Foundation Server.
