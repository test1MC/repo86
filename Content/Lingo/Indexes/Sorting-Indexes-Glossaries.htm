Sorting Indexes and Glossaries

How to Sort the Index or Glossary

  1. Open and translate the MadCap Flare glossary (.flglo) and index (.liindexmap) files in your Lingo project. See Translating Files.
  2. Select File > Project Properties. The Project Properties dialog opens.
  3. Select the Sort Order tab.

  4. From the Select file drop-down, select the index or glossary file you want to sort. The terms in the file appear in the grid below.

  5. Review the terms in each Entry column. These are the terms you entered in the MadCap Flare glossary (.flglo) and index (.liindexmap) files. If you need to enter a new term for sorting, type it in the Sort As column.

    You can also see the Entry and Sort As terms from the source project, if applicable.

    You can only make changes in the Sort As column. If you need to change the term in the Entry column, you can edit the translation in the glossary or index file.

  6. (Optional) Repeat steps 4-5 for any other index and glossary files.

When translated project is exported back to Flare, the index and glossary will use the new terms to alphabetize the entries.
