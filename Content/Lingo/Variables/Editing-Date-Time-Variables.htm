Editing Date/Time Variables

If you are translating a MadCap Flare project, it might contain variables that need to be translated. These are contained in a variable set file with an FLVAR extension. Sometimes an author might create what are known as "date/time variables," which consist of specifiers that automatically display a specific date and/or time.

How to Edit a Date/Time Variable

  1. Make sure the File List window pane is open.
  2. Double-click the variable set (such as MyVariables) that contains the variable you want to modify. This is a file with an FLVAR extension. The variable set opens in the Translation Editor.
  3. Click in a target segment. If it is a simple variable, you can just type the translation for it. However, if it is a special date/time variable, the Edit Format dialog opens.
