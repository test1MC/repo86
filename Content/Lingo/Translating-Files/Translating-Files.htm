How to Translate Files

Although you do not need to use TM when translating files, it is recommended that you do so. Therefore, before following the steps below, make sure you have performed the necessary setup tasks for TM. See About Translation Memory.

You can also set up your computer to use a machine translation (MT) service to provide additional translation suggestions. See About Machine Translation.
