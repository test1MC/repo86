Opening the Attribute Restriction Dialog

The Attribute Restriction dialog is part of the XML file filter. You can access the XML file filter when creating a new project or from the File Filters tool.

Creating a New Project

  1. Select File > New Project.
  2. Begin the process of creating a project. See Creating New Projects.

  3. On the second page of the wizard, click Add File. In the dialog that opens, select the XML file you want to add to the project.

  4. Create a new XML file filter. See About File Filters.
  5. To add a restriction, click Restriction.

Editing an Existing File Filter

  1. Select Tools > File Filters. The File Filters dialog opens.
  2. From the File type drop-down, select XML files.
  3. From the Filter drop-down, select the XML file filter you want to edit.
  4. Click Restriction.

Buttons and Sections

Include element if attribute value is in list below

If selected, the element will be translated if its attribute is listed.

Exclude element if attribute value is in list below

If selected, the element will not be translated if its attribute is listed.

Add Attribute Value

Enter an attribute to add to the inclusion/exclusion list. This can be any element; it does not need to be one already included in the XML file.

Adds an attribute to the inclusion/exclusion list.

Removes an attribute from the inclusion/exclusion list.

Clear Restriction

Removes the selected restriction from the filter.
