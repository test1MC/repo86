Column Mapping Dialog

If you are importing a CSV or Excel file, use the Column Mapping dialog to prepare your file for import to the termbase.

How to Open the Column Mapping Dialog

How to Use the Column Mapping Dialog

Buttons and Sections

Column

The column in your CSV or Excel file.

Term Property

Metadata attributes available in the termbase.

Reference

Languages that you can associate with each column.

Use first row as column headers

When this check box is selected, the first row of the CSV or Excel file is used as a header in the termbase concept grid.
