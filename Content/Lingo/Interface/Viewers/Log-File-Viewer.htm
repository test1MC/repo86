Log File Viewer

When you create a project from source files that contain errors, such as malformed XML, the Processing dialog opens. If you click Save Log, the file is saved and displayed in the Project Properties dialog. You can double-click a log file to see its details in the Log File Viewer.

Description

Displays a description of the log file.

Date

Displays the date and time that the log file was created.

Messages

Displays messages from the generation of the target that was built.

Warnings

Displays warnings that were encountered. These are "nice to know" issues, but not problems that will prevent you from working in and completing your project.

Errors

Displays errors that were encountered. A description and details of each error are given (e.g., the file, line, and character where the error was found).
