How to Back Up a File

  1. In the File List, open a topic.

  2. Select File > Send to Backups > Send to Backups.

    A message appears to confirm that the topic was copied to the Backups folder.

  3. In the message, click one of the following:

    • Yes Click Yes if you want to open the Backups window pane.
    • No Click No if you do not want to open the Backups window pane.
