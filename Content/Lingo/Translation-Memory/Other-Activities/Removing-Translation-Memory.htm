Removing Translation Memory

If you are no longer using a translation memory (TM) for a particular project, you can remove it, which simply hides that database from the list of available TMs in your project. If instead you want to delete a TM database permanently, see Deleting a Translation Memory.

How to Remove Translation Memory

  1. In the Translation Memory section, select Choose TM.
  2. Select the TM that you want to remove.
  3. Click Remove.
