Selecting a Language

See About Language Support.

How to Select a Language for a Collection

  1. Open the collection.
  2. In the Collection Editor, select the Language tab.

  3. Select the language from the list. The language must be displayed in bold.

How to Select a Language for a Standalone Movie

  1. Open the movie.
  2. Select Movie > Properties.
  3. Select the Language tab.

  4. Select the language from the list. The language must be displayed in bold.

How to Select a Language for an Object

  1. Open the movie.

  2. Double-click an object that contains text.

  3. Expand the Language section.

  4. Select the language from the list. The language must be displayed in bold.
