Copying Frame Backgrounds

How to Copy a Frame Background

  1. In the Frames window pane, select the frame whose background you want to copy.

    • Ribbon Select Frame > Copy Background.

    • Right-Click Right-click the frame and from the context menu, select Copy Frame Background.

  3. In the Frames window pane, select the frame whose background image you want to replace.

    • Ribbon Select Frame > Paste Background.

    • Right-Click Right-click the frame and from the context menu, select Paste Frame Background.
