Showing and Hiding Master Frames

You can show or hide master frame objects in the active frame. See Master Frames and Applying Master Frames.

How to Show and Hide Master Frames

  1. Open a frame that is using a master frame.
  2. Click the Master Frame(s) Visible drop-down at the top of the Frame Editor and choose one of the following:
    • None Select this if you want to hide all master frame objects.
    • Bottom Select this if you want only bottom-level master frame objects to be shown.
    • Top Select this if you want only top-level master frame objects to be shown.
    • All Select this if you want both top and bottom-level master frame objects to be shown.
