Removing Master Frames

Use the following steps to remove a master frame from a frame.

How to Remove a Master Frame

  1. In the Frames window pane, select the frame that you want to remove.

    • Right-Click Select Remove Master Frames.
    • Ribbon Select Frame > Properties.
    • Double-Click Double-click on the frame.
    • Right-Click Select Frame Properties.

  3. Expand the Master Frames section.
  4. Select No [Top Layer/Bottom Layer] Master Frame from the Top Layer or Bottom Layer field.
