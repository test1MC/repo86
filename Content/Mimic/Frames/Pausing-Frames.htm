Pausing Frames

How to Pause the Action in a Frame

You can add an object (e.g., button) to a frame and apply a "pause" action to it. If the user does not click the object in the output, the movie continues to play without interruption. If the user clicks the object, the movie pauses until the object is clicked again.

  1. Open the movie.
  2. Open the frame.
  3. Add the object (e.g., button, shape) that you want to use to pause the action.
  4. Double-click the object.
  5. Expand the Action section.
  6. Select Pause.

How to Pause the Display Time for a Single Frame in a Movie

You can disable (i.e., pause) the timing in one frame. This is useful if you add a button to a frame and want the user to click the button before the movie continues.

  1. Open the movie.
  2. Open the frame.
  3. In the Frame Editor, double-click the frame.
  4. Expand the Frame Transition tab.
  5. Select Pause movie at end of frame.

  6. Now you can add an object to the frame and apply an action to it (e.g., go to next frame) so that the user has a way to continue the movie. See Selecting Actions for Objects.

How to Pause the Display Time for All Frames in a Movie

You can disable (i.e., pause) the timing in all frames in a movie.

  1. Open the movie.

  2. Select Movie > Properties.

    The Movie Properties dialog opens.

  3. Select the Transition tab.
  4. Select Pause movie at end of frame.

  6. Now you can add objects to any frames in the movie and apply actions to them (e.g., go to next frame) so that the user has a way to continue the movie. See Selecting Actions for Objects.
