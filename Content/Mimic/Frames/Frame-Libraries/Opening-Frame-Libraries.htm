Opening Frame Libraries

How to Open a Frame Library

  1. Select View > Frame Libraries.
  2. In the local toolbar of the Frame Libraries window pane, click the drop-down containing the name of the current frame library .
  3. From the list, select the frame library that you want to open. The frame library opens, displaying the frames contained within it.

What's Next?

After you open a library, you can add frames to it or use frames from it. See Adding Frames to a Library and Using Frames from a Library.
