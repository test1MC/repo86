Using Frames From a Library

How to Use a Frame From a Library

  1. Open a movie.
  2. Open a frame library.
    • Local Toolbar In the Frame Libraries window pane, click on a frame that you want to add. In the local toolbar, click .
    • Right-Click Right-click on the frame and select Add Frame to Movie.

    The frame is added as the last frame of the movie. If you want to move the frame, open the Frames window pane and drag the frame to the desired position.
