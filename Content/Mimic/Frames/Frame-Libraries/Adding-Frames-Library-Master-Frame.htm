Adding Frames From a Library to a Master Frame

How to Add a Frame to a Master Frame

  1. Open a movie.
  2. Open a frame library.
    • Local Toolbar In the Frame Libraries window pane, click on a frame that you want to add to the master frame. In the local toolbar, click .
    • Right-Click Right-click on the frame and select Add Frame to Master Frame.

    The frame is added to the Master Frames window pane and it opens in the Master Frame Editor, which is nearly identical to the regular Frame Editor, except that it has a blue background. You can use this editor to make any changes to the master frame.
