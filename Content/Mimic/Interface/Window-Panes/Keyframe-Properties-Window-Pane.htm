Keyframe Properties Window Pane

The Keyframe Properties window pane opens by default on the right side of Mimic. This window pane is dynamic and changes depending on the type of keyframe you have selected. If the Object, Sound, Effect, or Frame Properties window pane is open, it will change to the Keyframe Properties window pane as soon as you click on a keyframe.

How to Open the Window Pane

  1. Open a movie that contains one or more objects or object effects that contain keyframes.
  2. Select View > Timeline.
  3. In the Timeline window pane, click Frame View or Action View, depending on the object type.
  4. Right-click on a keyframe (diamond) and select Properties.

Tasks You Can Perform in This Window Pane

  • Edit Keyframes You can edit the interpolation, or transition, of the object or clip effect from one keyframe state to the next. You can also use the window to edit values particular to each keyframe type. See Editing Keyframes.
  • Move Keyframes You can move individual keyframes to new starting times in the Timeline window pane by using Time field. See Moving Keyframes.
