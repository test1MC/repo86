Select Window

This window lets you select from a list of open programs when you record a movie or create a movie from a single image (using the "Application" or "Window" method).

How to Open This Window Pane

  1. Initiate a recording either by recording a new movie or frame, or by creating a new screen capture. See Recording Movies, Recording New Frames and Creating Movies from Screen Captures.

    Mimic minimizes and a rectangle with a red border appears on your screen, along with a task bar.

  2. In the task bar, click the down arrow and from the drop-down menu, select either Snap to Window or Move Window into Rectangle.

    The Select Window opens.
