Effect Properties Window Pane

The Effect Properties window pane opens by default on the right side of Mimic. This window pane is dynamic and changes depending on the type of effect you have selected. If the Object, Sound, Keyframe, or Frame Properties window pane is open, it will change to the Effect Properties window pane as soon as you click on an object's effect bar.

How to Open the Window Pane

  1. Open a movie that contains one or more objects that contain effects.
  2. Select View > Timeline.
  3. In the Timeline window pane, click Frame View or Action View, depending on the object type.
  4. Right-click an object's effect bar and select Properties.

Tasks You Can Perform in This Window Pane

  • Edit Effects You can edit the effect's position, size, and mask type (if applicable). See Editing Effects.
  • Add Effect Keyframes You can add keyframes to clip effects, changing the position and size of the clip effect area. See Adding Keyframes.
  • Edit Effect Keyframes You can edit clip effect keyframes. Editing gives you access to interpolation, or the transition effects between keyframes. See Editing Keyframes.
