Sound Properties Window Pane

The Sound Properties window pane opens by default on the right side of Mimic. From here, you can edit the name and timing of audio objects that have been added to a movie. If the Frame, Object, Effect, or Keyframe Properties window pane is open, it will change to the Sound Properties window pane as soon as you click on an audio object.

How to Open This Window Pane

  1. Open a movie that contains one or more audio files.
  2. Select View > Timeline.
  3. In the Timeline window pane, click Frame View.
  4. Right-click on an audio object and select Properties.

Tasks Can Perform in This Window Pane

  • Name You can rename the audio object by entering a new name in the Label field.
  • Time Span You can select a new start time for the audio object by entering a new time in the Start field.
  • Edit You can edit an audio file. This opens the Sound Editor. See Sound Editor.
  • Loop You can choose to loop the audio so it will play continuously for the length of the frame by selecting Loop.
