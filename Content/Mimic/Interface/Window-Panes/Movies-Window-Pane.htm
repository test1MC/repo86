Movies Window Pane

This window lets you see a snapshot of the first frame for each movie that you have added to a collection. You can easily open any movie in the collection by double-clicking the appropriate snapshot image in the Movies window pane.

Following are the primary tasks that you can perform in this window pane.

  • Creating Blank Movies
  • Importing Movies
  • Creating Movies from PowerPoint Presentations
  • Opening Movies
  • Collection Editor
