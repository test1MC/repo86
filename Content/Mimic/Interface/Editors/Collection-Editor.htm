Collection Editor

You can create standalone movies or movie collections. The Collection Editor is used as a central control area for all of the movies contained within a collection. The Collection Editor has a local toolbar, allowing you to easily add movies to your collection. There are also multiple tabs in the editor, used for various purposes.

Collection Editor Local Toolbar

You can use the local toolbar to manage your movie collection directly from the Collection Editor.

Sections

The Sections tab lets you modify movie and collection details, such reordering the movies within the collection and renaming movies and files.

Language

See Selecting a Language.

Variables

You can create variables for collections. The variables you create here are available to all movies within the collection. See Creating Variables.

Conditions

You can create your own condition tags. Condition tags created here are available to all movies within the collection. See Creating Condition Tags.

After applying condition tags to the appropriate elements, you can tell Mimic to include or exclude those conditions in the output. See Associating Condition Tags.

Targets

You can create as many targets as you need. You can use different settings for each target, so that each output is unique.

  • Creating Targets
  • Editing Targets

Project Link

Linking a collection to a Flare project lets you automatically have access to any variables or condition tags found in that Flare project. See Linking to Flare Projects.
