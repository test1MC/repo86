Recording Options Dialog

Use this dialog to set options pertaining to the recording mode, sound, callouts, and cursors before you record a movie. You can do the following in this dialog.

  • Automatically Adding Callouts to Recorded Movies
  • Selecting the Callout Style for Recording
  • Automatically Adding Cursors to Recorded Movies
  • Selecting the Cursor Style for Recording
  • Enabling and Disabling the Magnifier
  • Selecting the Recording Mode
  • Enabling and Disabling Sound for Recording
  • Selecting the Audio Input Source
  • Adding Typing Box Shapes for Recording
  • Enabling and Disabling Window-Based Recording Optimization
