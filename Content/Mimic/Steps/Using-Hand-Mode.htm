Using the Hand Mode

The hand mode in the local toolbar of the Frame Editor lets you drag areas of a frame around. This is useful when you have zoomed in on a frame so much that you cannot see all of it in the editor.

How to Use the Hand Mode Feature

  1. Open a frame.

    • Ribbon Select Home> .
    • Keyboard Shortcut Hold down the space bar to activate hand mode. To exit hand mode, release the space bar.
    • Right-Click Right-click anywhere in the Frame Editor, and from the context menu, select Hand Mode.
  3. Click on the frame and drag your mouse to move the frame so that you can see hidden portions of it.
