Using Select Mode

Select mode lets you select objects in the Frame Editor. Once an object is selected, you can view or edit its properties, apply an effect to it, rotate or resize it, and edit it in a variety of other ways.

How to Use the Select Mode Feature

  1. Open a movie.

    • Ribbon Select Home> .
    • Right-Click Right-click anywhere in the Frame Editor, and from the context menu, select Select Mode.
  3. In the Frame Editor, click the object you want to select.
