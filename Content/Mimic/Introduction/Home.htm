Get Started

Welcome to MadCap —a software simulation tool that lets you create instructional movies easily and quickly, all without needing to have any programming knowledge. Following are some of its primary characteristics:

  • Movie output in HTML5 and PDF
  • A robust set of features for editing movies
  • Live preview of movie or frame with synchronized audio
  • Unique features such as condition tags and palettes
