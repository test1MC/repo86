Adding Graphics to Frames

There are various graphics that you can add to a movie, such as bubbles, loops, arrows, and cursors. All graphics can be found in the Timeline window pane's Frame View.

How to Add Graphics to Frames

  1. Open a movie.

    • Ribbon Select Home> Objects > Graphics.

    • Frame Editor In the local toolbar of the Frame Editor, select Objects > Graphics.

  3. Select the appropriate option:

  4. Click on the frame and drag the mouse to draw the shape, then release the mouse button when you are finished.
