Adding Audio as Objects

How to Add Audio as an Object

  1. Open the frame.

  2. Select Home > Insert Sound.

    The Add Sound to Frame dialog opens.

  3. To Select Existing MP3 or WAV File From Sound Library

    1. If you have already imported or recorded an audio file, click the drop-down on the left side of the dialog and select Sound Library.
    2. In the field below, select an audio file.

    To Select System Sound

    1. Click the drop-down on the left side of the dialog and select System Sounds.
    2. In the field below, select an audio file.

    To Import MP3 or Wav File Into Sound Library

    1. Click Import new sound.
    2. In the dialog that opens, find and double-click the audio file.

    To Record New MP3 File

    1. Ensure that a recording device is connected to your computer.
    2. Click .
    3. When you are finished recording, click .
    4. (Optional) In the dialog that opens, you can type a new name for the sound.
    5. Click OK.

  4. (Optional) Click to listen to the file you selected in the Add Sound to Frame dialog.

  5. Click OK. The sound is added to the frame as an object, as indicated by the Sound bar in the Timeline's Frame View option.

You can also add audio to cursors. In addition, you can add audio to a movie and to individual frames in a movie. See Adding Audio to Cursors, Adding Audio to Movies, and Adding Audio to Frames.

What's Next?

After you add an audio object to a frame, you can adjust its start and end time by dragging the black bar in the Timeline window pane's Frame View option.
