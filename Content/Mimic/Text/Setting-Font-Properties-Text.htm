Setting Font Properties for Text

After you add text to an object, you can adjust the font properties (e.g., font size, color).

How to Set Font Properties

    • Keyboard Shortcut Press F2 to highlight the text, then click the highlighted text to place the cursor in the text area.

    • Double-Click Double-click the object. In the , expand the Text section.
  2. Select the text to which you want to apply the formatting changes.

  3. Select the Home ribbon and make changes using any of the options:

    Click this button to apply bold to the selected text.

    Click this button to apply italics to the selected text.

    Click this button to underline the selected text.

    Click this button to remove the formatting from the selected text.

    Click the down arrow to choose a color to apply to the selected text. To see advanced color options, select More colors.

    Choose the font family for the selected text.

    Choose a font size for the selected text.

You can also align text using the buttons in the Home ribbon. See Aligning Text.
