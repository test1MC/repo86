Adding Padding

How to Add Padding to All Frames in a Movie

  1. Open a movie.

  2. Select Movie > Properties.

  3. Select the Appearance tab.

  4. In the Padding section, enter numbers in the Left, Right, Top, and/or Bottom fields to set the width of the padding (in pixels).

    You can also use the Copy Down button to enter the padding values more quickly. After you enter a number in the first padding field (Left), you can click this button to automatically enter the same number in the rest of the fields.

  5. The padding is added to the frames in the movie, displayed in the default background color of the frame.

How to Add Padding to an Object

  1. Open a frame that has an object added to it.
  2. Double-click the object (not the frame itself).
  3. In the , expand the Text section.

  4. In the Padding section, enter numbers in the Left, Right, Top, and/or Bottom fields to set the width of the padding (in pixels).

What’s Noteworthy?
