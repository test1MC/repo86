Enabling and Disabling Window-Based Recording Optimization

If you are using a newer operating system (e.g., Microsoft Windows 10 or higher), you may find that windows are cut off when using the "Snap to Window" or "Move Window into Rectangle" recording options. This can be remedied using a feature in the Recording Options dialog. This feature is only recommended if you are using a newer operating system. Enabling this feature while using an earlier operating system may cause issues when recording.

How to Enable or Disable the Recording Optimization Option

  1. Select File > Recording Options.

    The Recording Options dialog opens.

    You can also open this dialog from the Options button at the bottom of the recording area. 

  2. In the Options section, click Optimize record region to add or remove the check mark.
  3. Click OK in the Recording Options dialog.
