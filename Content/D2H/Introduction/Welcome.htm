Doc-To-Help Welcome

Welcome to MadCap —a single-source authoring tool that makes it possible to write in Microsoft Word once and publish to many different deliverables.

Key Benefits

Key Benefits

offers the following:

  • Authoring in familiar Word environment
  • Access to all Doc-To-Help features without leaving the Word interface
    • Doc-To-Help menu under the File menu
    • Ribbons in Word to handle Doc-To-Help, project, and target functionality
    • Doc-To-Help project panel to easily manage all of your project elements
  • Quick and easy creation of online Help, training manuals, user guides, knowledge bases, and more
  • Publishing to multiple outputs, including responsive web, PDF, EPUB, and more
  • Professional built-in output themes, with ability to customize your own
  • Multi-language support, through integration with MadCap Lingo
  • Team authoring features
