Doc-To-Help Ribbon

Doc-To-Help Ribbon

The following commands are available in the Doc-To-Help ribbon when it is opened in Full mode (see Dynamic Add-In Modes).

Rename or delete an existing topic in a document, or add a new topic. See Adding, Renaming, and Deleting Topics. See the online Help or the Documents, Templates, and Content Guide.

Apply the Heading 1, 2, 3, 4, or Body Text style to the selected text in the source document. See About Styles.See About Styles. See the online Help or the Design Guide.

Create a hotspot link to topics containing the selected keywords, to a group of topics, to a bookmark in a document, or to a link tag. See Creating Links. See the online Help or the Design Guide.

Create a hotspot that will display additional text on the same line, in a dropdown, or in a popup. See Creating Expanded, Drop-Down, or Popup Text. See the online Help or the Design Guide.

Designates a hotspot as an index keyword. See Creating Index Keywords in a Source Document. See the online Help or the Design Guide.

Designates a hotspot as a member of a group. See Adding a Topic to a Group. See the online Help or the Design Guide.

Designates a hotspot as a link target. See Adding Link Tags. See the online Help or the Design Guide.

Sets a subset of the topic properties. See Setting D2HML Topic Properties. See the online Help or the Design Guide.

Used to mark text as conditional by platform, target, attribute, or any combination of all three. See Marking Text as Conditional. See the online Help or the Design Guide.

Creates a hotspot to a text or rich content variable. See Inserting a Variable. See the online Help or the Design Guide.

Creates a section that is either collapsed or expanded by default. See Creating an Expanding/Collapsing Section. See the online Help or the Design Guide.

Removes custom formatting created in the selected text. See Clearing Doc-To-Help Markup Language Styles. See the online Help or the Design Guide.

Apply a Doc-To-Help style on the selected text. See About Doc-To-Help Markup Language. See the online Help or the Design Guide.

Insert a note or image in the left margin of a printed target. The margin note can be displayed as a popup in online outputs if desired. See Creating Margin Notes. See the online Help or the Documents, Templates, and Content Guide.

In online outputs, use the Image Map Editor to create hyperlinks within an image. See Creating Image Maps. See the online Help or the Documents, Templates, and Content Guide.

You may insert multimedia (video and audio) into online outputs. See About Multimedia . See the online Help or the Documents, Templates, and Content Guide.

Insert HTML Help ActiveX Controls into your Word documents. See Inserting an HTML Help ActiveX Control. See the online Help or the Documents, Templates, and Content Guide.

Insert cross-references that will appear as page numbers in printed outputs and hyperlinks in online outputs. See Inserting Cross-References. See the online Help or the Documents, Templates, and Content Guide.

Add terms to your glossary from any Word source document. See Adding Terms to the Glossary. See the online Help or the Documents, Templates, and Content Guide.

View the current Doc-To-Help version number.
