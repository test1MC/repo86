Source Documents

What You Can Add in Source Documents

Following is a list of elements and objects that you can add to your source documents. We're not listing the many elements and objects that are built in to Word (e.g., images, lists, tables). For information about adding those kinds of features, see the Help provided with Word. Instead, we are focusing on the elements and objects that are specific to .

Additional Tasks

In addition to adding certain elements and objects, you can perform the following tasks when it comes to Word source documents:

  • Convert DOC Files to DOCX See Converting DOC Files to DOCX.
  • Convert to HTML5 You can convert a Word document that is already part of your project to HTML5. The file extension of the converted document will be .xml. The primary reason for converting Word documents to HTML5 is to then import your Doc-To-Help project to MadCap Flare. See Converting Documents to HTML5.
  • Set Document Properties The Document Properties dialog is used to view or change the properties of a document. See Setting Document Properties.
  • Find Projects Containing a Source Document See Finding Projects Containing a Source Document.
 Notes

Doc-To-Help styles reside in individual Doc-To-Help templates (such as D2H_NORM.dotx), so you should create your files using a predefined Doc-To-Help template and edit it to your specifications. See About Templates and Styles.

For more on using the styles in the Doc-To-Help ribbons, see Applying Styles in Microsoft Word.
