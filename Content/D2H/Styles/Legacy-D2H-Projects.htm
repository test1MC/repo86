Legacy Doc-To-Help Projects Using HTML5 and CSS

If you have legacy Doc-To-Help projects, your HTML5 and CSS files will continue to be supported. Also, you still have the ability to select, add, edit, and remove CSS files from those projects. You can do this from the Help Targets dialog (see Editing Target Settings) or Project Settings dialog (see Setting Project Properties). Also see Editing a CSS and Style List Dialog.

If you prefer to use a tool where you can edit and work with HTML and CSS files, you might consider converting to MadCap Flare. See madcapsoftware.com.
