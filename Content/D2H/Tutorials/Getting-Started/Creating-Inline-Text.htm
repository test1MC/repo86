Creating Inline Text

Inline text is expanding, drop-down, or popup text. It is a great way to add dynamic information to your online output.

Steps

  1. Go to the second paragraph in the All About San Diego topic and select the phrase America's Finest City.

  2. In the Doc-To-Help ribbon, click Inline Text.

  3. Choose Expand text inline, and in the text box, add the following:

    Stay classy, San Diego!

  4. Click OK.

    Doc-To-Help might inform you that this is an invisible style. This means the inline text we added will be invisible in this document (our source document), as well as in our target, until clicked. Since that is okay to keep the inline text invisible in the source document for this tutorial, click No to close the message if it appears.

    When we build the target, "America's Finest City" will be a hyperlink that displays "Stay classy, San Diego!" when clicked.

  5. Save the document.
