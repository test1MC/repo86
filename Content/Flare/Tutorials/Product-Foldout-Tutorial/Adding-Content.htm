Adding Content to Topics

Adding your own content to the topics is relatively simple. But because the output is print-based, and a bit unique, there are a few things to keep in mind as you work in the XML Editor.

Layout Modes and Style Mediums

The way a topic is displayed in the XML Editor depends on the layout mode that is selected, as well as the medium. This is especially important with this template, because you really need to see the appropriate page layout as you add content. If you are looking at a topic in Web Layout mode, you won't see this and it will be more difficult to work.

How to Select the Correct Layout Mode

  1. Open the Content Explorer and double-click Front2.htm.

    Chances are pretty good that your topic will initially be displayed in Web Layout mode with the default medium, like this:

  2. Click the Layout (Web) drop-down and select Print Layout.

    The look of the topic changes to this:

    This is the Back page layout (not the one you created, but the original page layout that came with the template). It is being shown because it is the first page layout listed alphabetically in the Content Explorer.

    But what you really need is the Front2 page layout that you created.

  3. In the local toolbar click the Page Layout drop-down and select Front2.flpgl, and in the flyout menu select Normal (the name of the only page in that layout).

    The display changes, showing the correct page layout:

    Also, notice that the medium selected is print. (You might need to click in the topic first to see this change.)

    This happened automatically when you switched from Web Layout mode to Print Layout mode. Flare anticipated that you might create both online and print-based output from your project, and the default medium is typically used for online output while the print medium is used for print-based output.

  4. In the Content Explorer, double-click Back2.htm.

  5. If the topic displays in the wrong mode, click the Layout (Web) drop-down and select Print Layout.

    The topic displays with the original Back.flpgl page layout. It looks correct, but lets sync it with the new page layout you created.

  6. In the local toolbar click the Page Layout drop-down and select Back2.flpgl, and in the flyout menu select Normal (the name of the only page in that layout).

Adding Content in the Front2 Topic

Now we need to add content to the topics. This includes text, images and variables. We will start with the Front2 topic.

In this template, we've created some variables. On the first panel of the foldout, we used them for the year of publication and company name. Then on the last panel we used a variable for the product foldout title. Variables can be beneficial if you plan to use the same page layout for multiple outputs. For one target you can use the default definitions for the variables, and you can override the variable definitions for any of the other targets you might generate (see Overriding Variable Definitions in Targets). For more about variables and overriding definitions in targets, see the online Help.

How to Add Content for the First Panel in the Front2 Topic

  1. In the XML Editor, click the Front2.htm topic tab to bring it to the front.
  2. In the Front2 topic, replace the heading with the text H1 General Information. Then replace the paragraph below it with different text. Don’t make it too long, but a few lines is okay; you can copy text from the original Front.htm topic if you like.
  3. After the new paragraph text, press ENTER to create a new empty paragraph.
  4. In the Content Explorer, expand the Resources folder, then expand Images under it.

  5. Click the Squares.png file, then drag and drop it to the new empty paragraph in the XML Editor.

  6. Click after the new image, and press ENTER to create a new empty paragraph.
  7. In the local toolbar, click the Character button .
  8. Find and select the copyright © character. Click Insert, then Close when finished (if you used the dialog).
  9. Type a space after the copyright character that was inserted. Then in the local toolbar click (the face of the button, not the down arrow). The Variables dialog opens.

  10. With the General variable set selected on the left side, choose the Year variable.

  11. The variable is inserted into the topic.
  12. Type a space after the variable that was inserted. Then in the local toolbar click again. The Variables dialog opens.

  13. With the General variable set selected on the left side, choose the CompanyName variable.

  14. The variable is inserted into the topic.

  15. After that variable, press ENTER again. You can then enter another line of text if you want to match what was done in the original Front topic. When finished, the first panel of your Front2 topic should look something like this:

How to Add Content for the Third Panel in the Front2 Topic

The text that you just added will display in the first panel of the output. The background image will display by itself as the second panel. Now we want to start adding content for the third panel.

  1. After the last line of text in the first panel, press ENTER. Then type H2 Lorem Ipsum, which will be a second-level heading.

  2. Press ENTER again and enter some paragraph text so that it takes up a couple of lines (again, you can copy text from the original Front.htm topic if you like).

    Notice that the new text moves past the image and into the third panel automatically. That’s because the first text that you added, as well as the image, are using the first body frame, which is twice the width of the other two body frames. The right margin in the first body frame is so wide that it doesn’t allow any text to display over the background image. So when you add more text than can fit in the first body frame, it starts using the second body frame, which as far as the end user is concerned, appears as the third panel.

  3. Select the new heading and paragraph text, then copy and paste it a couple of times. When you are finished, it might look something like this:

How to Add Content for the Fourth Panel in the Front2 Topic

In the fourth panel of the front topic, we want to add a company logo, followed by a variable that displays the foldout title. However, since we have not yet applied styles, the new content will not yet be placed where they should.

  1. After the last line of text in the third panel, press ENTER.
  2. In the Content Explorer, expand the Resources folder, then expand Images under it.

  3. Click the Logo.eps file, then drag and drop it to the new empty paragraph in the XML Editor.

  4. Click after the new image, and press ENTER to create a new empty paragraph.
  5. In the local toolbar click (face of the button, not the down arrow). The Variables dialog opens.

  6. With the General variable set selected on the left side, choose the Title variable.

  7. The variable is inserted into the topic.

Adding Content in the Back2 Topic

Now let’s move on to the Back2 topic, where we have a similar situation in that one of the panels has a background image with color. However, in this case, we want to add some content to that panel so that it displays on top of the background image.

How to Add Content for the First Panel in the Back2 Topic

  1. In the XML Editor, click the Back2.htm topic tab to bring it to the front. The first panel contains the background image and default text.

  2. The h1 text currently has the same color as the background image, so you cannot even see the text. However, if you select it, you will temporarily see the text while it is highlighted.

  3. Replace the heading and the paragraph under it with the following paragraphs:

    • First paragraph Get Started in
    • Second paragraph Just a Few Steps
    • Third paragraph For more help or troubleshooting, please visit our online documentation:
    • Fourth paragraph www.yourcompany.com/help
    • Fifth paragraph Created Using MadCap Flare

    When you are finished, it should look something like this (if you once again highlight the heading):

How to Add Content for the Second Panel in the Back2 Topic

  1. After the last line of text in the first panel, press ENTER.
  2. In the Content Explorer, expand the Resources folder, then expand Images under it.

  3. Click the Step1.png file, then drag and drop it to the new empty paragraph in the XML Editor.

  4. Click after the new image, and press ENTER to create a new empty paragraph.
  5. Type Step 1 and press ENTER.

  6. Type a short two-line paragraph. When you are finished, the foldout might look something like this:

    All of the content is still in the first panel, but later when we apply styles, this will change. In an actual project, you probably would apply styles as you add content, but because we want to treat styles separately in this tutorial, we’re going to do all of that later.

How to Add Content for the Third Panel in the Back2 Topic

  1. After the last line of text created for the second panel, press ENTER.
  2. In the Content Explorer, expand the Resources folder, then expand Images under it.

  3. Click the Step2.png file, then drag and drop it to the new empty paragraph in the XML Editor. The image displays at the top of the second panel, but that will change later.

  4. Click after the new image, and press ENTER to create a new empty paragraph.
  5. Type Step 2 and press ENTER.

  6. Type a short two-line paragraph. When you are finished, the foldout might look something like this:

  1. After the end of the paragraph that you typed, press ENTER.
  2. In the Content Explorer, expand the Resources folder, then expand Images under it.

  3. Click the Step3.png file, then drag and drop it to the new empty paragraph in the XML Editor.

  4. Click after the new image, and press ENTER to create a new empty paragraph.
  5. Type Step 3 and press ENTER.

  6. Type a short two-line paragraph. When you are finished, the foldout might look something like this, with the most recent text at the top of the third panel:
