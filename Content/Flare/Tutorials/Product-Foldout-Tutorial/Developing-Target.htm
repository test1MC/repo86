Developing a Target

The project you created already contains one PDF target.

We are going to create a new PDF target that is not necessarily identical to this existing target, but the same in the most important areas.

How to Create a Target

  1. In the Project Organizer, right-click the Targets folder, and from the context menu select Add Target. The Add File dialog opens.
  2. The only factory target template should already be selected, so in the File Name field at the bottom of the dialog, type Foldout-New.

  3. In the Output Type field, select PDF.

  4. Click Add. The new target is created and opens in Flare.
  5. In the Target Editor, select the General tab.

  6. In the Primary TOC field, select TOC2, the TOC you created.

  7. Select the Advanced tab.

  8. Remove the check marks from the following fields: Generate TOC proxy, Generate index proxy, and Generate glossary proxy.

  9. Select the PDF Options tab.
  10. In the Title field, give your foldout a title.
  11. In the Author field, enter your name.
  12. If you want, you can select Include Crop and Registration Marks.

  13. Select Convert RGB Colors to CMYK.

  14. Scroll down on the tab, and in the Navigation field, select Page Only.
