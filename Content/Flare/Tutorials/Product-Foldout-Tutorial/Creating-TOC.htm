Creating a TOC

At this point you've got two page layouts and two topics with similar names. Now we're going to tie them together so that the Front2 topic uses the Front2 page layout, and the Back2 topic uses the Back2 page layout. This is accomplished in a table of contents (TOC) file.

It might seem strange to be creating a TOC for a very short product foldout such as this. But remember, for print-based output (which is what you will be creating here), TOC files function as outlines, not as traditional TOCs.

How to Create a TOC

  1. Open the Project Organizer on the left side of Flare, and expand the TOCs folder. Notice the file under it called "TOC (Primary)." This is a TOC file—or, more accurately, an "outline TOC"—which was created for the existing Front and Back topics and page layouts. We're going to create another TOC just like this for the Front2 and Back2 files.

  2. Right-click the TOCs folder, and from the context menu select Add Table of Contents. The Add File dialog opens.

  3. The only factory TOC template should already be selected, so in the File Name field, type TOC2 as the name for the new TOC.

  4. Click Add. The TOC is added to the Project Organizer, and the TOC Editor opens to the right, displaying the TOC.

  5. Open the Content Explorer.
  6. Click the Front2 topic, then drag and drop it in the TOC Editor, above the two books. When you see a blue arrow, you will know that it can be dropped.
  7. Click the Back2 topic, then drag and drop it in the TOC Editor, below Front2.

  8. Click on each of the books and press Delete on your keyboard to remove them from the editor. By default, TOC entries display the topic title or first h1 text found. That’s why you see “Front2” and "Back2" as the labels for the topics. You can change the labels if you want, but it won’t matter in the output because remember, this is an outline, not a true TOC.

    What you've added to the TOC Editor are not the topics themselves, but rather a link to each topic. Now we've got to provide a link to the relevant page layouts.

  9. In the TOC Editor, double-click the Front2 entry. The Properties dialog opens.
  10. Select the Printed Output tab.

  11. In the Break Type field, select Page Layout Break.

  12. Click the Page Layout drop-down and select Front2.

  14. In the TOC Editor, double-click the Back2 entry. The Properties dialog opens.
  15. In the Break Type field, select Page Layout Break.

  16. Click the Page Layout drop-down and select Back2.
