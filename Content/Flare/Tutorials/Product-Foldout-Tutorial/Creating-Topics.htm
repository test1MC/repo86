Creating Topics

Now that you've created a front and back page layout, you can create a front and back topic to go with each.

How to Create New Topics

  1. In the Content Explorer, right-click the root Content folder, and from the context menu select New > Topic.

    The Add File dialog opens.

  2. Under the Factory Templates, select the NewTopic.htm template.

  3. In the File Name field, type Front2 as the name for the first new topic.

  4. Click Add. The topic is added to the Content Explorer, and the XML Editor opens to the right, displaying the topic.

  5. Follow the same steps above to create another topic, except name the second topic Back2. Now you have two new topics to go with the existing "Front" and "Back" topics.

    Currently, both new topics look rather plain when you open them. But that will soon change after you associate the topics with the two page layouts that you've already created. Also, additional content and some styles will help bring the topics together.
