Parts of a Style

A selector (e.g., h1, p) in a stylesheet includes a declaration made of two parts: properties and values. Examples of properties might be color, font-weight, or font-size. The values for those properties might be red, bold, and 14.0pt. See Important Style Terms and Concepts.

Getting to Know the Parts of a Style

Let's look at a style in true CSS code, and then how it looks in the stylesheet.

  1. From the Content Explorer, expand Resources > Stylesheets, and right-click the Styles.css file.
  2. From the context menu, select Open with, and then select Internal Text Editor. All the styles and properties from the Stylesheet Editor are displayed to the right in the Text Editor.

  3. Scroll down to roughly line 234 to view the h3 heading settings.

  4. From the Content Explorer, double-click Styles.css.

  5. From the Stylesheet Editor's local toolbar, make sure the first button is labeled View: Advanced.

  6. Filter to see heading styles, and select the h3 style element.

  7. From the local toolbar Show drop-down, select Show: All Properties.

  8. Expand the Font group. Notice how the style elements correlate between the Text Editor and the Stylesheet Editor. The h3 style (or selector) is paired with its own properties and values.

  9. From the Window ribbon, select Close All Documents.
