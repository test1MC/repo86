Setting Mediums in the Target Editor

You need to explicitly tell Flare which medium you want a particular target to use, but only if you intend to use a medium other than the defaults provided.

How to Set a Medium in the Target Editor

  1. From the Project Organizer, expand Targets, and double-click All-About-Austin-PDF.

  2. In the Target Editor, select the Advanced tab.

  3. From the Medium field, select print-black. This associates the medium with the target.

  5. Select the Project ribbon. Then from the Build Primary drop-down, select Build All-About-Austin-PDF.

    The Builds window pane opens at the bottom of the Flare interface, and the target begins to generate.

    When the build finishes successfully, the Build Progress cell turns dark green.

  6. Double-click the row to open the PDF output. Notice the h1 elements throughout the project, the xref links, and table header in the Music topic are black instead of in a color.

  7. Close the PDF.

  8. From the Window ribbon, select Close All Documents.
