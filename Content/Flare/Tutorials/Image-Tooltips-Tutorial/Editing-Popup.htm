Editing the Popup

Now that the tooltip works, you might want to make changes to the popup. Let's change the background color and maximum width of the popup.

How to Edit the Popup

  1. From the Content Explorer, expand Resources > Stylesheets.
  2. Double-click Styles.css.

  3. On the left side of the Stylesheet Editor, make sure the filter is showing all styles. Then find and expand (Generic Classes), and select the tooltip class.

  4. Under the Background group (if you are using group view), change the background-color property value to #cae4e9.

  5. Under the Unclassified group, change the max-width to 150px.

  7. Build and view the All-About-Austin-HTML5 target.

  8. In the output, open the Attractions topic, and hover over each of the images to see its tooltip. Notice the new background color, as well as the width of the popup.
