Accessing the LMS Package

Earlier in this tutorial, we chose settings in the HTML5 targets (eLearning tab) to integrate the output with an external learning management system (LMS). If you choose to do this with your own output, you need to access the resulting LMS package ZIP file, which can then be uploaded to your external LMS. Let's go find the package file that we generated for one of our HTML5 targets.

How to Access an eLearning Package

  1. From the Project ribbon at the top of Flare, click Open Output Folder.

  2. In Windows Explorer, you will see folders for each of the targets where you have generated output. Double-click the Combination-Course folder.

  3. Double-click the LMSPackage folder to open it.

  4. Notice the ZIP file, which has the name that we provided for it in the target (Austin Combination Course). This is what you upload to your LMS.
