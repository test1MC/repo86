Associating an Alias File With a Target

We're almost done, but not quite yet. A project can have multiple alias files and multiple targets. Even though we only have one alias file and one online target so far in this project, it's still a good idea to connect the two, because we might want to add more alias files and online targets in the future, and our current target needs to point to the correct IDs.

How to Associate an Alias File With a Target

  1. In the Project Organizer, expand the Targets folder and double-click All-About-Austin-HTML5.
  2. In the Target Editor, click the Advanced tab.

  3. Scroll down to the Alias File field, click the drop-down, and select My-Alias-File.
