Replacing Branding

When we generated and viewed the output earlier, the course displayed a few pieces that have to do with the branding of the course. These include the big picture (i.e., "hero" image) in the background on the first page, the logo, and the colors.

Let's replace these branding items with others.

How to Replace the Hero Image

  1. In the Content Explorer, expand the Resources > TemplatePages folder.

  2. Double-click Course-Title.flmsp to open that file in an editor to the right.

    This is a template page, which is used to automatically add content to topics in the output. In most cases, template pages are used by many topics to display the same content, but in this case the template page is used for just one topic, the first page of the course.

    This template page has three items in it.

    • First, there is a snippet that displays a rectangle header with a logo.

    • Second, there is a large rectangle image of the city of Austin, Texas with some text over it.

    • Third, there is a gray bar at the bottom, which is a proxy; this represents the regular content for each topic when the output is generated.

  3. We're going to replace the large image. But before we do that, let's update the text. Change Austin, Texas to San Diego.

  4. Right-click the div structure bar to the left of the hero image, and from the context menu select Style Class > Edit Style Class. This opens the Stylesheet Editor with the appropriate style (div.topic-hero) automatically selected.

  5. Make sure the following areas are selected in the Stylesheet Editor.

  6. To the right of the background-image field, click .

  7. In the Insert Image dialog, click at the bottom. Then, find and select the SD-Hero.png image that you saved to your computer in an earlier lesson, and click Open. Then click OK.

  8. In the Copy to Project dialog, click OK.

  10. In the Course-Title.flmsp file, notice the new background image.

How to Replace the Logo

  1. With the Course-Title template page still open, right-click the snippet at the top holding the blue rectangle with the logo. From the context menu select Open Snippet File.

  2. In the snippet file that opens, right-click the logo image and select Image Properties.

  3. In the Insert Image dialog, click at the bottom. Then, find and select the SD-Logo.png image that you saved to your computer in an earlier lesson, and click Open. Then click OK.

  4. In the Copy to Project dialog, click OK. The new image replaces the old one in the snippet.

How to Replace the Colors in the Stylesheet

  1. In the Content Explorer, expand Resources > Stylesheets, and double-click Styles.css to open the stylesheet.

  2. On the left side of the Stylesheet Editor, scroll up and expand Variables.

  3. Select :root. The properties on the right side changes to show some style variables set with various colors.

  4. To the right of the Brand1 field, click .

  5. In the Edit CSS Variable dialog, change the Value field to #F2774B.

  7. Use these same steps for Brand2, and change the Value to #FCF6F3.

How to Edit the eLearning Skin Component

Another place where we need to make some changes to get our branding colors is the eLearning skin component. This controls the look of the navigation buttons and progress bar at the bottom of the topics.

  1. In the Project Organizer, expand the Skins folder, and double-click eLearning to open it to the right.

  2. On the left side of the editor, select the Styles tab if necessary (depending on which tab was open the last time you were working in the editor, a different tab might initially be selected). You can now see the styles for the skin listed on the left and a preview on the right.

  3. We're going to use a trick to help us find the correct styles quickly. In the local toolbar, click Highlight.

  4. In the preview, we can see that there a couple of places using the blue from the template, but our main branding color is orange. So click the blue part of the horizontal bar from the progress bar in the preview. Doing this highlights the Inner Progress Bar area on the left.

  5. Expand Background.

  6. In the Color field, replace the hex number with #F2774B. The preview changes accordingly.

  7. The text in the progress bar would probably be a little hard to read if we change it to orange. So let's make it black. Expand Progress Text.

  8. Under that, expand Font.

  9. In the Color field, click the down arrow and select the black square. The preview changes accordingly.

  10. Scroll up in the styles area on the left side of the editor and expand Toolbar Button (hover).

  11. Under that, expand Background, and in the fields showing the blue hex number, replace it with #F2774B.

How to Edit the Test Results Skin Component

This project also has a test results skin component, which lets us control the look of the results page that displays at the end of the quiz. The only thing we're going to change in this skin component is the color of the heading that is at the top of that page.

  1. In the Project Organizer, under the Skins folder, double-click TestResults to open it to the right.

  2. On the Styles tab, expand Title. And under that, expand Font.

  3. In the Color field, replace the hex number with #F2774B. The preview changes accordingly.

How to Remove Other Unused Files

Let's get rid of a couple of more files that we won't need in this project.

  1. In the Content Explorer, click on Template Instructions.pdf and press DELETE. Then click OK.

    This PDF contains information about some of the things we've already done in this lesson, so we don't need it anymore.

  2. Under the Resources > TemplatePages folder, select Popups.flmsp and press DELETE. Then click OK.

    This template page was used for some popup topics that we removed earlier in this course. Since those topics were deleted, this template page is no longer necessary.
