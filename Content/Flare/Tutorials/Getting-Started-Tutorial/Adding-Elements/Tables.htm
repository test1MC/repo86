Inserting Tables

You can use tables for many purposes in Flare. We are going to insert a small table into a topic to compare some information.

How to Insert a Table

  1. Open the Introduction.htm topic.
  2. Scroll down to the History section of the topic. Add a new empty paragraph between the two paragraphs in this section.

  3. From the Insert ribbon, select Table (the face of the button, not the down arrow). The Insert Table dialog opens.

    If you click the down arrow below the Table button, it will open a grid that you can use to insert a basic table. That is fine for many uses, but for this tutorial, be sure to click the face of the button—not the arrow—so you can configure some properties in the Insert Table dialog.

  4. In the Table Size section, use the arrows to make the following changes:
    1. Set the Number of columns to 2.
    2. Set the Number of rows to 4.
    3. Set the Number of header rows to 1.

    As you change these settings, the table in the Preview section updates to show you how your table will appear in Flare.

  5. In the Table Style section, from the Table Style drop-down, select Alternate-Row-Color.css.

  6. Click OK. The empty table appears in the History section.

    Let's add some data to our table.

  7. Click in the upper-left cell of the table. This is your header row. Type Historical Affiliation.
  8. Click in the upper-right cell of the table. Type Date.
  9. In the Historical Affiliation column, type the following:
    1. In the first cell, type Spanish Empire.
    2. In the second cell, type First Mexican Empire.
    3. In the third cell, type United Mexican States.
    4. In the fourth (bottom) cell, type United States.
  10. In the Date column, type the following:
    1. In the first cell, type 1769–1821.
    2. In the second cell, type 1821–1823.
    3. In the third cell, type 1823–1848.

    4. In the fourth (bottom) cell, type 1848–present.

What are Table Styles?

Flare uses table styles to determine the look of a table. They are a useful way to make sure all of your tables have a consistent look. For example, you can use a table stylesheet to determine the colors for your table's rows and columns, or for its borders. You can apply the table style to one table or to all of the tables in your project. Table styles are handled separately from regular stylesheets. You can also use table styles to set default styles for the cell content. See Table Stylesheets.
