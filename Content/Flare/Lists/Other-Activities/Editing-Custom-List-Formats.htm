Editing Custom List Formats

After you create custom list formats, you can edit them in a couple of ways.

How to Use the Edit List Format Dialog

  1. Open a content file containing a custom format list.
  3. Click the outermost (i.e., first level) structure bar (ol or ul) for the list.
  4. Select Home > (down arrow) and choose Custom List Format > Edit Custom List Format.

  5. Use the Edit List Format make changes to the format.

  6. The dialog closes and any lists associated in your project with that custom format are automatically changed.
