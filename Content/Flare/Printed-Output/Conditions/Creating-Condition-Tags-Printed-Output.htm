Conditions for Print-Based Output

Conditions

Condition tags can be used to tell which content is to be used for some outputs and which is to be used for other outputs. provides you with some initial condition tags that are ideal for this purpose. However, if you want, you can always create additional condition tags or rename the default tags provided to meet your needs.
