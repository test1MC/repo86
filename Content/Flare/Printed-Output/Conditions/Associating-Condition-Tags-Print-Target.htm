Associating Conditions With a Print Target

After you apply the condition tags to the appropriate content for print-based output, you'll need to specify in a target (based on one of the print-based output types) which condition tags to include in the output and which to exclude.

How to Associate Conditions With a Print Target

  1. Open the target that you plan to build for your printed output. The Target Editor opens.
  2. Click the Conditional Text tab.

  4. By default, all tags will be included in the target unless you specify otherwise. If you want to exclude a condition from the output for this target, select the condition row and choose Exclude. If you want to make sure a condition is included in the output for this target, select Include.

    If you use the default condition tags provided by , you might select Include for the PrintOnly tag, and you might select Exclude for the Mobile, ScreenOnly, Tablet, and Web tags.

After you associate condition tags with your print target, you can move on to any of the other tasks for creating printed output. See Creating Print-Based Output.
