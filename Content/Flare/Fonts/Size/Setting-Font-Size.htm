How to Use a Style to Set the Font Size

  1. If Using Simplified View

      • In the local toolbar of the Stylesheet Editor, click the down arrow in the Font Size field and select a font size.

        OR

      • Select the Font tab. Click the down arrow in the Size field and select a font size.

      You can also type a number and a different unit of measurement directly in the field (e.g., 5in, 25px, 10em). See Units of Measurement.
