Editing Feedback Profiles

Unless you enable anonymous comments, every individual, even an administrator, must be a registered user of the output in order to add and reply to Feedback comments. At minimum, each registered user must enter a user name and email address. In the Skin Editor in (Community tab), you can specify additional required information and allow users to receive email notifications (see Specifying Fields for the Create Feedback Service Profile dialog and Displaying Feedback Notification Options to Users). As a user, you can register when you add your first comment.

All registered users can edit their existing profiles when necessary. There are a couple of ways to do this. First, both users and administrators can use the Edit User Profile shortcut button in the output to make any changes to a profile. Second, end users who are set up to receive email notifications can use the Unsubscribe link in emails to make changes to which notifications they receive.

How to Edit a User Profile Using the Shortcut Button (Users and Administrators)

  1. Open a topic in the output.

  2. In the toolbar click the Edit User Profile button .

    For the and outputs, this button is included by default, but you can use the WebHelp Toolbar tab in Standard skins, or the Toolbar tab in skins, to exclude it from the output. See Specifying Custom Toolbar Settings for WebHelp Output, Specifying Web Toolbar Custom Settings— Output, and Specifying Custom Toolbar Settings for HTML Help Output.

  3. In the dialog that opens, make changes to the fields as necessary.

    You do not need to select the notification check boxes in your user profile if you are also set up as an administrator. As an administrator you can be set up in Feedback Explorer to receive email notifications.

  4. Click Submit

How to Edit a User Profile Using Email (Users Only)

  1. If you previously selected to receive email notifications, you will see an Unsubscribe link in any emails you receive. Click this link.

  2. In the dialog that opens, make changes to the fields as necessary.

  3. Click Save.
