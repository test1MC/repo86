Get Started

As a new Flare user, it’s a good idea to familiarize yourself with the following information.

Key Concepts and Features

There are hundreds of features in Flare. But you don’t need to try to use all (or even many) of them. Become acquainted with the most important concepts and Flare’s key features to discover which ones might be particularly important to you.

Workspace

's workspace is flexible, uses a modern Multiple Document Interface (MDI), and gives you several options to work the way that you want. Get to know some of the more important parts of the workspace.

Architecture

Before starting your first project, you might want to explore ideas for developing your project architecture.
