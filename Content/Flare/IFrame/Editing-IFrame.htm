Editing IFrame

After you insert an IFrame element in a topic, you can always go back to it and edit it.

How to Use a Style to Edit the IFrame

  1. From the Content Explorer, open a stylesheet.

  4. Select the iframe style.

  5. Use the properties section on the right side of the editor to apply settings for the style. For example, you might want to control the IFrame size by editing the max-width and max-height properties, or add a border line around the IFrame by editing the border properties.

How to Use Local Formatting to Edit an IFrame

  1. Open the topic with the IFrame in it.

  2. Right-click the IFrame element, and from the context menu, select Edit IFrame.

  3. From the Insert IFrame dialog, do any of the following:

    • Edit Source field Update the URL that you want to display in the IFrame window.

    • Select Browse button This opens the Select IFrame Source dialog. Navigate to a source file (e.g., HTML file) to open.

    • Adjust Size Use the Width and Height fields to set custom values for the IFrame window.
