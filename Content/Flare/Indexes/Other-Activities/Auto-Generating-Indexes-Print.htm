Auto-Generating Indexes for Print

The following explains how to auto-generate an index.

How to Auto-Generate an Index

  1. Open a target.

  2. On the Advanced tab of the Target Editor, select Generate Index Proxy.

    This option is disabled by default for existing targets, but enabled for new targets. If an Index proxy is already present in a topic or snippet that is being included in the output, this auto-generate option has no effect.

When you generate the target, an index will be created and inserted after all of the other content, as well as after an auto-generated glossary, if you include one.
