How to Open Images in MadCap Capture for Editing

    • Open a topic where the image is inserted. Right-click on the image.

      OR

    • In the Content Explorer, locate and right-click the image file that you want to edit (e.g., Resources > Images > NameOfFile.png).
  2. From the context menu select Edit With MadCap Capture. The image opens in MadCap Capture.
  3. Edit the image in Capture. For details, see Capture's online Help.
  4. Save your work in Capture.

You can also make any image in your online Help interactive by linking it to another topic, bookmark within a topic, or an external file such as a website. See Inserting Image Hyperlinks.

If you are using the "generate web-safe images" option in targets, images that link to external locations, such as websites, are not affected by that option. See Generating Web-Safe Images.
