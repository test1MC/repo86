Setting the Cursor for Styles

In a stylesheet, you can specify the kind of cursor that is shown when users hover over a particular element, such as a cross-reference or a button image.

How to Set the Cursor for a Style

  4. On the left side of the editor, select the style that you want to edit (e.g., MadCap|xref, img, div.myclass).

  6. If you are using the group view, expand Extensions. In the field next to the cursor property, enter the kind of cursor you want to use (e.g., all-scroll, crosshair, grab, help, pointer).

    For more details about the cursor style property and available values for it, refer to the W3C and other resources on cascading stylesheets (CSS).
