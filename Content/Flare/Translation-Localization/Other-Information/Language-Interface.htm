Selecting a Language for the Interface

You can select one of the following languages for displaying the Flare interface: English, French, German, Japanese, or Chinese.

How to Select a Language for the Interface

  1. Launch Flare. The Select UI Language dialog opens.

    If you have previously selected to disable this dialog, you can enable it:

    1. Select File > Options. The Options dialog opens.
    2. Select the General tab.
    3. Select Show Select UI Language Dialog on Startup.
  2. Select the language.
  3. (Optional) If you do not want the dialog to display at launch, click Show this dialog on startup to remove the check mark.
  4.  The interface displays according to the language selection.
