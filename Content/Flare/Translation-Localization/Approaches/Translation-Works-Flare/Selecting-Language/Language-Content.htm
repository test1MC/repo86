Language for a Content Segment

Selecting language attributes at the content level can be used for organizing or identifying text for a certain language, but only if you are managing multiple languages in one project. When you apply a language attribute to a segment, it is flagged as containing content for the specified language.

How to Select a Language for Content

  1. Open the topic.
  2. Highlight the text where you want to change the language.
  3. Select Home > Language.
  4. From the Select Language dialog, choose a language. (If the language is displayed in bold, Flare sets the dictionary and spelling functions to the specified language.)
