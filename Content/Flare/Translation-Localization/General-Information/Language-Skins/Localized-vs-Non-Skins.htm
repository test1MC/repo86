Localized vs. Non-Localized Skins

provides completed language skins for certain languages, such as French, German, and Spanish. These languages are identified as being "localized" skins when you select a project or target language.

For other languages, you can create language skins so they can be applied to a project or target (see Adding Language Skins). After you create a language skin, it is identified as "localized." The language skins are stored in the Advanced folder of the Project Organizer.
