Authentication Information Dialog

This dialog opens when you are creating a ServiceNow® destination and you click Login Credentials. See Creating Destination Files for ServiceNow®.

  • Instance Enter the ServiceNow® instance (the part before service-now.com).
  • Client ID Paste the client ID.
  • Client secret Paste the client secret.

You can obtain the client ID and secret when you create an OAuth API endpoint for external clients in ServiceNow®. See Retrieving ServiceNow® Credentials.

After clicking Next, enter your ServiceNow® User name and Password. Then click Finish.
