Unlink Micro Content Dialog

This dialog opens if you right-click a micro content structure bar in a topic or snippet and select Unlink Micro Content.

  1. If do not want to keep the phrase at all in the Micro Content Editor, select Delete associated micro content phrase(s). If you do not select this option, the connection will be removed between the file and the phrase, but the phrase will remain in the Micro Content Editor, and the previously linked response will be converted to text.
