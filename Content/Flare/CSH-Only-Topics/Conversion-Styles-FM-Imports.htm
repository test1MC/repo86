Conversion Styles for FrameMaker Imports

The Import Styles Editor lets you specify how to convert each property of the FrameMaker styles that you are importing. If you do not enter a property value, the value from the FrameMaker document is used. If you enter a property value, it overrides the value from the FrameMaker document. This "Conversion Styles" option is used only if you have selected "Preserve FrameMaker Styles" when importing FrameMaker files. See Importing FrameMaker Files.
