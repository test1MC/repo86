Inserting Equations

See Equations.

How to Insert an Equation

  1. In a third-party editor, create your equation in either MathML or LaTex, and copy it.
  3. In the XML Editor place your cursor where you want to insert the equation.
    • Ribbon Select Insert > Equation.
    • Keyboard Shortcut Press CTRL+E.

    The Equation Editor opens.

  5. In the Text Editor area, paste your equation. In a moment, the Preview area will show the results.
  6. (Optional) In the Screen Tip field you can type a phrase that will appear when the end user hovers over the equation.

  7. (Optional) In the Alternate Text field you can type alternate text to display when the equation is not available, such as when a disabled individual is using a screen reader.

    For more information see Accessibility.

  8. The equation is added to the content file.
