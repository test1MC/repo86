Editing the Look of Micro Content

If you integrate micro content into your HTML5 search results, there are different ways you can control its look in an HTML5 skin.

It is also possible to use complex selectors in your stylesheet to make content look different in micro content search results than it does in topics in the output. See Single-Sourcing Micro Content Using Styles.

How to Edit the Look of Micro Content in a Skin

  1. Open an HTML5 skin.
  2. Select the Styles tab.
  3. In the local toolbar, make sure Web Medium is selected.

  4. On the left side of the editor, in the Search Results section, expand any of the following styles:

    Search Micro Content Result

    Search Micro Content Response

    Search Micro Content Response Expand

    Search Micro Content Response Collapse

    Search Micro Content Response Fade

    Search Micro Content Response Link

    Search Micro Content Response Link (active)

    Search Micro Content Response Link (focus)

    Search Micro Content Response Link (hover)

    Search Micro Content Response Link (visited)

    Search Micro Content Response Path
