Example—Replacing a Variable Tag's Attributes

You have a variable set called "General," which holds (among other things) a variable called "CompanyName." You want to find all the places in the project where this variable has been inserted into content files and replace it with another one. The new variable is called "Corporation" and is located in a different variable set called "Business."

Set the fields like this(you will need to click Add Row to enter the attribute):

  • Find
    • Tag
    • MadCap:variable
    • Attribute
    • name

    • General.CompanyName

  • Replace/Action
    • Set Attribute
    • name
    • Business.Corporation
  • Find in
    • (whole project)
  • File types
    • All Supported Files

Click Find All. After the results are all shown, and you confirm you want to change them, click Replace All.
