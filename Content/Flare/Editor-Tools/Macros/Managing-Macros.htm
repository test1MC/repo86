Managing Macros

How to Assign a Shortcut to a Macro

  2. Select the Tools ribbon. In the Macros section, click Manage. The Manage Macros dialog opens.
  3. From the Macro Name list, select the macro to which you want to assign a shortcut.

  4. Click Assign Shortcut.

  5. Click OK in the dialog, then use your keyboard to press the shortcut you want to use. The shortcut will be added to the grid on the left.

    If another command is already using the shortcut, tells you at the bottom of the dialog.

How to Remove a Shortcut From a Macro

  2. Select the Tools ribbon. In the Macros section, click Manage. The Manage Macros dialog opens.
  3. From the Macro Name list, select the macro from which you want to remove a shortcut.
  4. Click Clear Shortcut. The shortcut is removed.

How to Rename a Macro

  2. Select the Tools ribbon. In the Macros section, click Manage. The Manage Macros dialog opens.
  3. From the Macro Name list, select the macro you want to rename.
  4. Click Rename. The Rename Macro dialog opens.

  5. Enter a new name for the macro.

How to Delete a Macro

  2. Select the Tools ribbon. In the Macros section, click Manage. The Manage Macros dialog opens.
  3. From the Macro Name list, select the macro you want to delete.
  4. Click Delete. The macro is removed from the list of macros.
