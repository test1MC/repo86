How to Create Custom Date/Time Variables

  1. In the Project Organizer, expand the Variables folder and double-click a variable set. The Variable Set Editor opens.

  2. In the local toolbar click . A new row is added for the date/time variable.

  3. To enter a new name or comment, do one of the following:

    • Double-click in the appropriate cell and type the name or comment.

      OR

    • Click once in the appropriate cell and press F2 on your keyboard. Then type the name or comment.
