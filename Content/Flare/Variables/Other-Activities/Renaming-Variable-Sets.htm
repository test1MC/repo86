Renaming Variable Sets

You can rename a variable set in your project.

How to Rename a Variable Set

  2. Double-click the Variables folder. The variable sets in your project are displayed.
  3. Click on the variable set that you want to rename.
  4. Press F2 on your keyboard. The variable set name is highlighted.
  5. Type a new name for the variable set and press ENTER.
  6. If your file contains links to other files, you will be asked if you want to retain those links. If so, click Update Links.

For more information about renaming variables within a variable set, see Editing Variables.
