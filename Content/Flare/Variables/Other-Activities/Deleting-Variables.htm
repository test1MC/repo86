Deleting Variables

If necessary, you can delete a variable from the Variable Set Editor.

How to Delete a Variable

  2. Double-click the Variables folder. Your variable sets are displayed.
  3. Double-click the variable set (such as MyVariables) that contains the variable you want to delete. The Variable Set Editor opens to the right, with the variables page shown.
  4. Select the variable you want to delete.
  5. The variable is removed from the Variable Set Editor.
