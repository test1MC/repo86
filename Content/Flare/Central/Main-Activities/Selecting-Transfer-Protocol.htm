Selecting a MadCap Central Transfer Protocol

If you plan to bind a Flare project to a MadCap Central license, you can choose between a couple of transfer protocol options when performing the binding. This includes binding new projects or importing existing projects from Central. It does not affect existing projects that are already bound.

How to Select a MadCap Central Transfer Protocol

  1. Select File > Options. The Options dialog opens.
  2. Select the Source Control tab.
  3. Under MadCap Central Transfer Protocol, select one of the following:
    • SSH Also called “Secure Shell,” this is a network protocol that lets you communicate remotely and securely between computers.
    • HTTPS Hypertext Transfer Protocol Secure (HTTPS) is a more secure version of HTTP, which is the standard protocol for transferring data over the web.

    Some companies might prefer HTTPS, but SSH could be necessary for binding especially large projects to Central.
