eLearning Content Topics

When authoring in Flare, information is created in “topics.” The topics are created and arranged in the Content Explorer. They can be organized according to learning objectives, key concepts, practical applications, course types, etc. See Guidelines for Developing a Course.

When we talk about topics in the eLearning context, they are similar to terms such as learning blocks, content blocks, building blocks, learning modules, “chunks of training,” etc.

How to Create eLearning Topics

  1. Open a project.

  2. From the Content Explorer's local toolbar, click . This opens the Add File dialog to create or import a new topic. See Creating Topics.

  3. Replace the default text in the topic with learning information.

  5. Repeat steps 2 through 4 for as many eLearning topics as needed.

In the Content Explorer, you can create custom folders and subfolders. This can help you organize related content.
