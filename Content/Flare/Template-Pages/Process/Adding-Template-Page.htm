Adding a Template Page File to a Project

How to Add a Template Page File to a Project

    • (Recommended) Right-Click In the Content Explorer, right-click a folder and from the context menu select New > Template Page.

    • Ribbon Select Project > New > Add Template Page.

    The Add File dialog opens.

  2. In the File Type field at the top, make sure Template Page is selected.

  5. In the File name field, type a new name for the template page.
  6. (Optional)   See Applying Conditions to Content.
  7. (Optional)   See Associating Tags with Files.
  8. Click Add. The template page file is added. The XML Editor opens to the right, with the new template page shown.
