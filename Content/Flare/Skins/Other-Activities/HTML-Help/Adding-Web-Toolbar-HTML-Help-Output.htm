Adding a Web Toolbar to HTML Help Output

See Specifying Web Toolbar Settings—HTML5 Output.

How to Add a Web Toolbar to HTML Help Output

  1. Open a Standard skin.
  2. Select the HTML Help Setup tab.
  3. Click Display toolbar in each topic.

As an alternative, you can add a web toolbar with the help of a topic toolbar proxy. This lets you add a toolbar in specific places within topics, as well as limit the topics that have the toolbar. See Inserting Topic Toolbar Proxies.

Make sure that you associate the skin with the target. See Associating Skins with Targets.
