Importing Skins

Not only can you add a new skin, but you can also import an existing skin (FLSKN file).

How to Import a Skin

    • Ribbon Select Project > New > Skin.

    • Right-Click In the Project Organizer, right-click on the Skins folder and from the context menu select Add Skin.

    The Add File dialog opens.

  3. Find and select the FLSKN file that you want to import.
  4. Click Open. The Source File field now contains the path to the file that you are importing. Also, the name of the file is displayed in the File Name field.
  5. If you want to give the skin a different name than that for the imported file, click in the File name field and replace the text.
  6. (Optional)   See Applying Conditions to Content.
  7. (Optional)   See Associating Tags with Files.
  8. Click Add. The skin is added and opens in the Skin Editor.

After you import a skin, you can edit its settings and then associate it with a target. See Editing Skin Settings and Associating Skins with Targets.
