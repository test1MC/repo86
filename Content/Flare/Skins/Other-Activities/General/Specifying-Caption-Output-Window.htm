Specifying the Caption for the Output Window

How to Specify the Caption for the Output Window

  1. Open a Standard or HTML5 Tripane skin.
  2. Select the General tab.
  3. In the Caption field, type the caption just as you want it to appear in the output window.
  4. Click somewhere else in the Skin Editor.

Make sure that you associate the skin with the target. See Associating Skins with Targets.
