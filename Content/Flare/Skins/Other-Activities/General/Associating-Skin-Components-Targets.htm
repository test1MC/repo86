Associating Skin Components With Targets

See Skins and Proxies.

How to Associate a Skin Component With a Target

  1. Open an HTML5 target.
  2. Select the Skin tab.

  3. In the Component Default Skins section, click the drop-down arrow in any of the fields (Menu, Topic Toolbar, eLearning Toolbar, Search Bar, Search Results, Favicon, Central Account, Test Results) to select the skin component that you want to associate with the target.
