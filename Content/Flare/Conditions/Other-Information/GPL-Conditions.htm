Global Project Linking and Conditions

With Global Project Linking, you can import content and project files contained in another project, thus allowing you to maintain the information in one location but reuse it in any other project. See Global Project Linking.

In addition to specifying certain files or file types to include or exclude from the import, you can go a level deeper through the use of condition tags. If you have applied condition tags to files in your project, you can use the Import Conditions field to tell to include or exclude certain files, depending on the condition tags that are applied to them at the file level. Simply click the Edit button to choose the conditions. In the Conditional Text dialog for this feature, you can use the Basic or Advanced mode for creating condition tag expressions.
