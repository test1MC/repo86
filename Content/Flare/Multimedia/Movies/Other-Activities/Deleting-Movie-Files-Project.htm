Deleting Movie Files From the Project

Use the following steps to remove a movie (SWF) file from a project. Movie files are added to the project automatically when you insert a movie into a topic.

  2. Expand the Resources folder, and double-click the Images folder to open it (unless you have stored these files in a custom folder).
  3. Locate and select the SWF file that you want to delete.
  5. You may be asked if you are certain you want to send the file to the recycle bin. If so, click Yes.

If you want to remove a movie from a specific topic, see Deleting Movies from Topics.
