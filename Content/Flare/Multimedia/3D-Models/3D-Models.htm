3D Models

If you want to enhance your output using 3D, supports Universal 3D (U3D) files. These files allow you to show users an interactive 3D model in print or web output. You can insert 3D models directly into your files.

The U3D content above is displayed as an image file to ensure compatibility across all devices.

Some browsers do not support U3D files. This is because certain browsers do not support the plug-ins required to play this file type.

If you select a U3D file outside the project, that file is then copied and placed inside the project. The U3D file is stored in the Resources > Multimedia folder of the Content Explorer.
