Editing Destinations

If necessary, you can edit the settings for a publishing destination that you have created.

How to Edit a Publishing Destination

  1. From the Project Organizer, expand Destinations and double-click the publishing destination that you want to edit.
  2. Complete the fields in the Destination Editor. The options that you see depend on the transfer type. See the following for field details of each type:
    • Creating Destination Files for Elasticsearch
    • Creating Destination Files Using File Transfer Protocol
    • Creating Destination Files Using File System
    • Creating Destination Files Using Secure File Transfer Protocol
    • Creating Destination Files Using Source Control
    • Creating Destination Files for Salesforce®
    • Creating Destination Files for ServiceNow®
    • Creating Destination Files for Zendesk
