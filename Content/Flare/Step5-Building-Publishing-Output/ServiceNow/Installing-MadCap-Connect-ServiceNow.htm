Installing MadCap Connect for ServiceNow®

To publish output to ServiceNow®, you must have MadCap Connect for ServiceNow® selected during the installation process. In a typical installation, it should already be included. If it wasn't selected during the initial process, you can modify your Flare installation via the Control Panel in Windows. In the wizard that opens, select Modify and then enable MadCap Connect for ServiceNow®.
