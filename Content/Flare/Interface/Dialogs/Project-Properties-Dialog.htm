Project Properties Dialog

This dialog is used to see basic information (e.g., location of files, creation date) and to make property changes for your Flare project in the following ways.

  • Primary Table of Contents Read more…   See Associating a Primary TOC With a Project.
  • Primary Page Layout Read more…   See Associating Primary Page Layouts With Projects.
  • Primary Stylesheet Read more…   See Associating Primary Stylesheets With All Files.
  • Language Read more… You can select a language to use for the project. See Selecting a Language.
  • Source Control Read more… You can select several options related to source control. The options shown are different, depending on which source control provider your project is bound to. See Source Control.
    • Bind Project Read more… If you have a Flare project open, you can "bind" (i.e., connect) it to a source control application (see Binding a Project to Source Control). Binding a project to source control lets you take advantage of the many multi-author features available through source control integration. This includes automatic check-in, check-out; viewing differences between source control and local copies of the project files; and more. See Source Control.
    • Unbind Project Read more… You can remove the connection between the Flare project and the source control application. See Unbinding a Project from Source Control.
    • Check In Files Read more… You will see this option if your project is bound to Microsoft Team Foundation Server. See Checking In, Committing, and Submitting Source Control Files.
    • Commit Read More… See Checking In, Committing, and Submitting Source Control Files.

    • Submit Read More… You will see this option if your project is bound to Perforce Helix Core. See Checking In, Committing, and Submitting Source Control Files.

    • Check Out Files Read more… You will see this option if your project is bound to Microsoft Team Foundation Server or Perforce Helix Core. See Checking Out Source Control Files.
    • Lock Files Read more… See Locking Files.
    • Unlock Files Read more… See Unlocking Files.
    • Get Latest Version Read more… You will see this option if your project is bound to Microsoft Team Foundation Server or Perforce Helix Core. See Updating or Getting the Latest Version of Source Control Files.

    • Update Read More… See Updating or Getting the Latest Version of Source Control Files.

    • Undo Check Out Read more… You will see this option if your project is bound to Microsoft Team Foundation Server. See Reverting or Undoing a Checkout of Source Control Files.
    • Revert Read more… You will see this option if your project is bound to Git, Perforce Helix Core, or Subversion. See Reverting or Undoing a Checkout of Source Control Files.
    • View Differences Read more… See Viewing Differences in Source Control Files.
    • View History Read more… See Viewing the History of Source Control Files.
