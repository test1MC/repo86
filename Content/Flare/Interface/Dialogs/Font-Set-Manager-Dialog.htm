Font Set Manager Dialog

You can perform the following actions from this dialog:

  • Select a Font Set Use this dialog to find and select a font set. See Selecting Fonts.
  • Create a Font Set Use the buttons in this dialog to open the Define Font Set dialog, where you can create a new font set. See Creating Font Sets.
  • Edit a Font Set Use the buttons in this dialog to open the Define Font Set dialog, where you can edit an existing font set. You can edit font sets after you create them if you want to add or remove fonts from the font set. You can also change the order of fonts in the set if you want to give a different font a higher or lower priority in the font set. See Editing Font Sets.
  • Delete a Font Set If you do not need a font set anymore, you can delete it from the Font Set Manager. See Deleting Font Sets.
