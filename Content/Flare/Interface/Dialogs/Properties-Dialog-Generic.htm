Properties Dialog (Generic)

From the Properties dialog, you can access properties for certain content file types, such as topics, stylesheets, and images. You can see general properties for the file(s), such as location, size, and more. You can also associate file tags or conditions with the file(s).

How to Open This Dialog

  1. Select the file(s) in the Content Explorer or File List window pane.
