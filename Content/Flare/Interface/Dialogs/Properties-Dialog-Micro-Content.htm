Properties Dialog (Micro Content)

This dialog is used to specify various settings for a micro content phrase.

How to Open This Dialog

  1. Open a micro content file.
  2. On the left side of the editor, right-click a phrase.
  3. From the context menu select Properties.

Phrase Properties Tab

Use this tab to control various properties related to a micro content phrase.

Phrase ID

Lets you edit the ID for the micro content phrase, which is used when you create a context-sensitive Help (CSH) ID for the phrase in the Alias Editor. See Editing Micro Content Phrase IDs and Creating and Assigning Identifiers.

Phrase Title

Lets you edit the title for a micro content phrase. This title can be used if you create field-level CSH and want to control the text that displays in the browser tab that opens as a result. See Editing Micro Content Phrase Titles.

Include micro content in search results

Lets you include or exclude the micro content phrase's response from the search results. See Including or Excluding Micro Content Phrases in Search.

Conditional Text Tab

Use this tab to apply condition tags to the selected micro content phrase. See Conditions on Phrases.

Micro Content or Snippet Conditions Tab

Use this tab to apply micro content conditions or snippet conditions to the selected micro content phrase. See Micro Content Conditions and Using Snippet Conditions.

Micro Content or Snippet Variables Tab

Use this tab to apply override variables for the selected micro content phrase. See Single-Sourcing Micro Content Using Variables and Overriding Variable Definitions in Snippets.
