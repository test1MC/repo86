Mail Setup Dialog

This dialog can be used to specify your outgoing email server and your email address. This is used as part of the file review process, letting you email files to other individuals. When you first attempt to send a topic for review, will search for your email server and enter it automatically. However, you can open this dialog whenever you want in order to make changes to the server or your email address. See Reviews and Collaboration and Sending Files for Review in Contributor or Flare.

How to Open This Dialog

  1. Open a topic.

  2. Select Review > Send For Review.

    The Send Topic for Review Wizard opens.

  3. Select Send a review package file.
  4. Click Next.
  5. Click Setup Mail.

Buttons and Sections

Outgoing Mail Server

Displays your outgoing email server.

Your E-mail Address

Enter your email address, which others will see when you send them topics for review.

Lookup in Active Directory

Searches for your server and enters it into the Outgoing Mail Server field.

If you are using 64-bit Flare, it will work with 64-bit Office. If you are using 32-bit Flare, it will work with 32-bit Office.
