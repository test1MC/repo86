Enter Custom Attribute Dialog

In you can set attributes for a tag by using the Attributes window pane. Attributes let you provide additional information about tagged content and how that content should behave.

You can use the "dir" attribute to change the base direction of a block of text. The default value for the "dir" attribute is "ltr" (left-to-right), but you can change the value to "rtl" (right-to-left). If you apply the "rtl" value to tagged content, that content will start at the far right edge of the window (as opposed to the left) and move to the left as the amount of content increases. You would set this attribute by clicking on an area of tagged content, clicking in the right column next to dir, and selecting rtl from the drop-down.

This dialog is for advanced users who are familiar with tags and attributes. It lets them create a new attribute and provide a value for it.
