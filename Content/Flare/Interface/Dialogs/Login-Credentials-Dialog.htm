Login Credentials Dialog

Use this dialog when publishing output to enter the user name and password required for accessing the server to which you are publishing. If you do not enter the user name and password ahead of time (via the Destination Editor), another dialog will open later when you actually publish the output, asking for the user name and password.

How to Open This Dialog

  2. Double-click the Destinations folder. The destination(s) in your project are displayed.
  3. Double-click the destination that you want to open. The Destination Editor opens.
  4. In the Type field, select FTP or Source Control.

  5. Complete the Host Name field.

    OR 

    Complete the Source Control section.

  6. Click Login Credentials.

Buttons and Sections

Server

Displays the server where the output is being published.

User name

Enter the appropriate user name to access the server. (You may need to check with your network administrator.)

Password

Enter the appropriate password to access the server. (You may need to check with your network administrator.)
