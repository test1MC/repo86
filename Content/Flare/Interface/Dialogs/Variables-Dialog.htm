Variables Dialog

This dialog opens whenever you attempt to insert a variable into your content. You can perform the following actions from this dialog:

  • Insert a Variable Select the variable you want to apply to your content. See Inserting Variables.
  • Edit Variables Open the Variable Set Editor, where you can create new variables or edit existing variables. See Editing Variables.
  • Pin Variables Select a recently used variable from the Recently Used Variables group and pin it so you can use it again later. See Pinning Variables.

Pinned Variables

You can pin your favorite variables so you can easily find them again later. This is a way to limit the list of variables to those that you use the most.

Recently Used Variables

After you insert a variable, it is added to a group of your 20 most Recently Used Variables. You can pin any variable found in this group.
