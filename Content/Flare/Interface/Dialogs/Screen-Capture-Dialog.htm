Screen Capture Dialog

This dialog is used when incorporating an image with Capture—MadCap's screen capture and editing tool. Using this dialog, you can do the following.

  • Image (Add) You can capture an image on your screen and add the image file to the project (without inserting the image into a topic). See Adding Screen Capture Files to Projects.
  • Image (Insert) You can capture an image on your screen and insert it into a topic at the same time. See Inserting Screen Capture Images.
  • Launch MadCap Capture You can simply open the MadCap Capture application from .
  • Show Magnifier When performing a screen capture, you can turn on a magnifier, which shows a close-up portion of the region as you draw it. This helps you grab a more specific area.

In order to use this feature, you must have MadCap Capture installed on your computer.

For more information about screen captures and editing images, see Capture's online Help.
