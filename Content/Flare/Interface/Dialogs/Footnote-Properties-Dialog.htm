Footnote Properties Dialog

You can use this dialog to accomplish several editing tasks for footnotes, including the following.

Main Activities
  • Editing Footnote Comment Font Settings
  • Editing Footnote Comment Position
  • Editing Footnote Comment Spacing
  • Editing Footnote Number Formats
  • Editing the Look of Footnote Numbers
  • Restarting Footnote Numbering

For more information, see Footnotes and Editing Footnotes.
