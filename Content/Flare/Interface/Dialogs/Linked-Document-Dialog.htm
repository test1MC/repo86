Linked Document Dialog

This dialog opens if you try to save content that is linked to other project files, Word documents, or FrameMaker source documents that you imported.

After you import Word or FrameMaker files to , you should make future changes to the content either in or in the source files, but not in both. The dialog serves as a warning if you have initially indicated that you want to make future changes in the source files (maintaining the link between the source files and ), but now you are trying to edit in the editor.

How to Open This Dialog

Imports From Other Projects

  1. Import files from another project. See Global Project Linking—Importing Files from Other Projects.
  2. Open a file that you imported and attempt to make a change.
  3. In the dialog that opens, click Continue with operation.
  4. Save your work.

Imports From Word or FrameMaker

  1. Import Word or FrameMaker documents. When doing this, make sure you select Link Generated Files to Source Files.
  2. Open a topic created from the import.
  3. Make a change in the topic.
  4. Save your work.

Buttons and Sections

Link

Displays the source file.

Open

Opens the source file.

Save the document

Saves the document. The link to the source document still exists, but only the document contains the changes that you made. The source file does not contain the changes.

Save the document and remove the link

Saves the document and removes the connection to the source file. This means that you will not be prompted with this dialog when making future changes to the document.

Save the document using a new name

Opens a dialog, which lets you save the document under a new name. This means that the connection will be maintained between the original source files and the project, but you will not lose the information that you added in because a new file is created containing that information. However, the new file will not be linked to the source files.
