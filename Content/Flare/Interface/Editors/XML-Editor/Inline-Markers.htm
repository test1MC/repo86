Inline Markers

You can show or hide markers for all inline elements in the active topic or document. You might find this feature useful if you have imported DITA file content. The reason for this is that DITA inline tags are often for semantic purposes, rather than for changing the look of content. By enabling the inline markers feature, it is easy to tell when particular content is using a tag, especially when that tag does not necessarily change the presentation of the content.
