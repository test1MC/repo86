Reviewed Snippet Editor

This editor can be used to view a reviewed snippet. From this editor, you can also accept a snippet from it so that it is copied to the source file in the project. See About Reviews and Contributions.

How to Open This Editor

  1. Select Review > File Reviews.

    The File Reviews window pane opens.

  2. Make sure Inbox is selected at the top of the window pane. All reviewed files that you have brought into your project, but have not yet been accepted, are listed in the pane.
  3. Double-click one of the rows containing a snippet file.

Buttons and Sections

Opens the source snippet in the XML Editor.

Lets you accept the reviewed snippet. After accepting the snippet, it overwrites the source snippet in your project. Any edits or annotations (comments) in the reviewed snippet therefore become part of the original snippet. See Accepting Reviewed Files.
