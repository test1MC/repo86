Formatting Window Pane

The Formatting window pane can be used to work with both styles and local formatting in the open content file.

Style Inspector and Local Style Properties

This window pane consists of two tabs—Style Inspector and Local Style Properties.

  • Style Inspector See Style Inspector.

  • Local Style Properties See Local Formatting.

How to Open This Window Pane

  • Ribbon Select Home > Formatting Window.
  • Keyboard Shortcut Press CTRL+F12.

Local Toolbar

Selects the content for the parent tag and displays its style and properties in the window pane.

Local Toolbar—Style Inspector

Lets you create a new style class based on the current formatting.

Local Toolbar—Local Style Properties

Show: [Properties]

  • Set (Locally) Properties
  • Assorted Relevant Properties
  • All Properties
