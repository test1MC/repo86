File List Window Pane

How to Open This Window Pane

  • Ribbon Select View > File List.
  • Keyboard Shortcut Press CTRL+R.

What You Can Do in the File List Window Pane

Following are the tasks that you can perform in the File List window pane. For more details and steps for each, see the online Help.

  • 3D Models (Insert) Read more… You can click on a U3D file in the File List window pane and drag it to the XML Editor. This inserts a 3D model into that file. See Inserting 3D Models.
  • Audio (Insert) Read more… You can click on an audio file in the File List window pane and drag it to the XML Editor. This creates an audio object. See Inserting Audio Files.
  • Browse Sequence Entries (Add) Read more… You can click on a topic file in the File List window pane and drag it to the Browse Sequence Editor. This adds an entry in the browse sequence to that topic. See Creating a Browse Sequence.
  • Column Information (Customize) Read more…  See Customizing Column Information.
  • Condition Tags (Apply) Read more… You can apply condition tags to files. See Applying Conditions to Content.
  • Cross-References (Insert) Read more… You can click on a topic file in the File List window pane and drag it to the XML Editor. This creates a cross-reference to that topic. See Inserting Cross-References.
  • Data (Export) Read more… You can export all or selected data from the File List window pane to a comma-separated value (CSV) file. This can be quite useful since there are several columns of data in the window pane. See Exporting File Data to a CSV File.
  • File Tags (Associate) Read more… You can associate tags with files. See Associating Tags With Files.
  • Images (Insert) Read more… You can click on an image file in the File List window pane and drag it to the XML Editor. This inserts the image. See Inserting Images.
  • Language (Select) Read more… You can select a language to be used. See Selecting a Language.
  • Movies (Insert) Read more… You can click on a movie file in the File List window pane and drag it to the XML Editor. This inserts the movie. See Inserting Movies.
  • Read Only Read more… You can mark files as "Read Only" so that they cannot be modified.
  • Related Topics Link (Insert) Read more… You can click on multiple topic files in the File List window pane and drag them to the XML Editor. This opens the Insert Related Topics Control dialog, which lets you create a related topics link. See Inserting Related Topics Links.
  • Snippets (Insert) Read more… You can click on a snippet file in the File List window pane and drag it to the XML Editor. This inserts the snippet. See Inserting Snippets.
  • Source Control (View Checked Out or Modified Files) Read more… If you are using Git or Subversion, you will see files you have modified. See Viewing Files that are Checked Out or Modified.
  • Stylesheets (Apply) Read more… You can apply a stylesheet to topics, snippets, and template pages. See Associating Stylesheets Locally With Specific Files.
  • Table of Contents Entries (Add) Read more… You can click on a topic file in the File List window pane and drag it to the TOC Editor. This adds an entry in the table of contents to that topic. See Creating a TOC.
  • Title (Add) Read more… You can specify a title for a topic. See Changing Topic Titles.

User

Title

Displays the title for the topic. This title is used in the following situations:

  • Help Control Links The title displays as a selection when users click Help control links (i.e., related topics links, concept or "see also" links, and keyword links) that are associated with that topic.
  • Window Title Bar If the topic is opened in a new window or outside of the project, the display title is shown in the title bar of that window.

Description

Displays the meta description for the file. See Adding Meta Descriptions for Topics.

Class

Displays the topic style class associated with the topic. The properties for this style class will be applied to the entire topic. For more information see Styles and Stylesheets.

Stylesheet

Displays the stylesheet associated with the file.

Conditions

Displays the condition tags associated with the file.

Language

Displays the language associated with the file.

Dimensions

Displays the size of image files.

Resolution

Displays the resolution of image files.

Searchable

Indicates whether the topic is to be included in searches or not. This setting can be adjusted in the Properties dialog for the topic.

[File Tags]

If you create file tag sets, these will be displayed in columns with the tag names associated with each file. See File Tagging.

[Preview Bar]

This is the area at the bottom of the File List window pane. When you click on a file, this bar shows its name, size, and last modified date.
