Content Explorer

The Content Explorer (along with the Project Organizer) is one of the most often used areas of the interface. Just as it sounds, the Content Explorer is used to hold all of the content-related items in your project—both topics and non-topic files.

How to Open the Content Explorer

  • Ribbon Select View > Content Explorer.
  • Keyboard Shortcut Press CTRL+J on your keyboard.

Where Each Type of File is Stored

You can store new files anywhere you like in the Content Explorer, either in the root Content folder or in any subfolders under it. See Creating Subfolders.

We recommend that you store non-topic content files in subfolders under Resources.

What You Can Do in the Content Explorer

Following are the primary tasks that you can perform in the Content Explorer:

  • 3D Models (Insert) Read more… You can click on a U3D file in the Content Explorer and drag it to the XML Editor. This inserts a 3D model into that file. See Inserting 3D Models.
  • Audio (Insert) Read more… You can click on an audio file in the Content Explorer and drag it to the XML Editor. This creates an audio object. See Inserting Audio Files.
  • Cross-References (Insert) Read more… You can click on a topic file in the Content Explorer and drag it to the XML Editor. This creates a cross-reference. See Inserting Cross-References.
  • Folders (Add) Read more… In order to keep your topics and other files organized and easier to find, you can create subfolders in the Content Explorer and move topics into them. See Creating Subfolders.
  • Images (Insert) Read more… You can click on an image file in the Content Explorer and drag it to the XML Editor. This inserts the image. See Inserting Images.
  • Images (Add/Open) Read more… You can open an image that you have added to your project. When you open the image, it displays in the Image Viewer within . See Images, Adding Images to Projects, and Opening Images.
  • Micro Content (Add/Open) Read more…  See Micro Content and Creating Micro Content.
  • Movies (Insert) Read more… You can click on a movie file in the Content Explorer and drag it to the XML Editor. This inserts the movie. See Inserting Movies.
  • Page Layouts (Add/Open) Read more…  See Page Layouts, Adding Page Layout Files, and Opening Page Layouts.
  • Properties Dialog (Open) Read more… You can open the properties dialog for individual content items by selecting them and clicking . From the Properties dialog you can perform a variety of tasks, depending on the type of file you selected.
  • Snippets (Add/Open) Read more…  See Snippets, Adding Snippet Files, and Opening Snippets.
  • Snippets (Insert) Read more… You can click on a snippet file in the Content Explorer and drag it to the XML Editor. This inserts the snippet. See Inserting Snippets.
  • Stylesheets (Add/Open) Read more… A stylesheet is a file with a .css extension (cascading stylesheet) that holds styles, properties, and values. You need to apply a stylesheet at the project, target, or file level if you want to use the styles in the associated topics, snippets, or template pages. See Regular Stylesheets, Creating Stylesheets, and Opening Stylesheets.
  • Table Stylesheets (Add/Open) Read more… This is a stylesheet that lets you create and use different styles that pertain specifically to tables in your project. Using a table stylesheet makes it easier to have multiple looks for tables throughout your project. See Table Stylesheets, Creating Table Stylesheets, and Opening Stylesheets.
  • Template Pages (Add/Open) Read more…  See Template Pages, Creating Template Pages, and Opening Template Pages.
  • Topics (Add/Open) Read more… See Topics, Creating Topics, and Opening Topics.

[Preview Bar]

This is the area at the bottom of the Content Explorer. When you click on a folder or file, this bar shows its name, size, and last modified date.

You can drag and drop files from Windows into the Content Explorer. See Dragging and Dropping Files Into Projects.

You can quickly insert a text hyperlink to a file (e.g., topic, image, snippet) in the Content Explorer. To do this, click and drag the destination file from the Content Explorer to the location in the topic where you want to insert it. As you drag the file into the topic, a vertical red bar acts as a guide to help you place the link in the topic. When you release the mouse button, the Insert Hyperlink dialog opens. You can modify any fields in the dialog (if necessary) and click OK to insert the hyperlink.

If you are working with HTML Help and import a RoboHelp project that has been merged with other CHM files or includes shortcut controls, the linked files are placed in a special subfolder in the Content Explorer (Resources > CHMSupport).
