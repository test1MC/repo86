Index Window Pane

See Indexes.

How to Open This Window Pane

  • Ribbon Select View > Index Window.
  • Keyboard Shortcut Press F9.

What You Can Do in This Window Pane

Following are the primary tasks that you can perform in this window pane.

  • Create Index Keywords and Subkeywords There are two ways to link a topic to an index keyword using this window pane. First, you can open the topic (or a snippet), place your cursor where you want to insert the keyword, and use the top area of the Index window pane to create the keyword (see Inserting Index Keywords). Second, you can also use the bottom area of the Index window pane to add index keywords or subkeywords (see Adding Index Keywords) and assign topics to them (see Assigning Topics to Index Keywords).
  • Create Index Links  See Creating Index Links.
  • Remove  See Deleting Index Keywords.
  • Rename See Editing Index Keywords.
