File Reviews Window Pane

This window pane lets you work with review documents (topics and snippets) that you have received from others.

Inbox and Sent Topics

There are two different views in the window pane. The information in the window pane changes, depending on the view you select.

  • Inbox This view shows files that reviewers have returned to you.
  • Sent Files This view shows topics and snippets that you have sent for review.

To switch from one view to another, click the drop-down field at the top of the window pane and make your selection.

Toolbars

This window pane can be used for either Central file reviews or for review packages. Some of the options are available for only one process or the other. Following are the options that you will see in the Inbox and Sent Files views.

Status

Indicates the status of file in the Central review process:

  • Submitted The file has been submitted by all reviewers on Central. The next step is for the author to close out the review process for the file.
  • Closed The review process for the file has been closed, which means no more edits can take place for the file on Central, unless the author sends it out again for review. After a file is closed, it can then be opened in Flare so that the changes can be accepted or rejected. After that, the file can be accepted back into Flare, replacing the original source file.
