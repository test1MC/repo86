Inserting QR Codes

How to Insert a QR Code

  2. In the XML Editor place your cursor where you want to insert the QR code.

    • Ribbon Select Insert > QR Code.

    • Keyboard Shortcut Press CTRL+SHIFT+Q.

    The Insert QR Code dialog opens.

  4. Select the General tab.

  5. From the Content type field, select one of the following options.

    Text

    URL

    Email Address

    Contact Information

    SMS

  6. (Optional) Click in the Size field.

  7. (Optional) If you want to apply a specific style class to the QR code, you can select it from the Style Class field.

  8. (Optional) In the Screen Tip field you can type a phrase that will appear when the end user hovers over the QR code.
  9. (Optional) In the Alternate Text field you can type alternate text to display when the QR code is not available, such as when a disabled individual is using a screen reader. For more information see Accessibilitythe online Help.
  10. (Optional) Use any of the other tabs to provide additional settings for the QR code.

    • Position tab  See Using Float Positioning.

      You can also use absolute position on QR codes via the context (right-click) menu. See Using Absolute Positioning.

    • Borders & Margins tab  See Adding Borders to QR Codes, Adding Margins to QR Codes, and Adding Padding to QR Codes.

    • Background tab  See Adding Backgrounds to QR Codes.
  11. The QR code is inserted.

This also works for QR codes, which are converted to images in the output. See Applying Empty Alt Text to Images.
