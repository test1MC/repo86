Inserting Text Popups

A text popup is a link that opens a popup box containing basic text that you provide. Example…This is an example of a text popup.

Inserting text popups requires only a few simple steps

How to Insert a Text Popup

  2. In the XML Editor, highlight the content that you want to use as the link (or "hotspot") to open the popup.

  3. Select Insert > Text Popup.

    The Insert Text Popup dialog opens, with the text that you highlighted already shown in "The Hotspot Text" field.

  4. In The Popup Text field, type the content for the popup.
  5. The link for the text popup is added to the topic. You can change the appearance of the link by modifying the MadCap|popupHead and MadCap|popupBody styles in the Stylesheet Editor.

You can also create a popup that opens a topic or external file, as opposed to opening only basic text. For more information see Inserting Topic Popups.

You can use the built-in analysis tool and reports to find broken links. See Viewing Broken Links and About Reports.
