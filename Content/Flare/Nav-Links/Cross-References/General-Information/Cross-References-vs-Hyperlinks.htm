Cross-References vs. Text Hyperlinks

Use the following to help determine when to use a cross-reference and when to use a hyperlink.

Cross-References

Text Hyperlinks

Use for linking to other topics within the project, because the link text will automatically update if changes are made to the destination topic heading, file name, etc.

Not designed for linking to content outside of the same project

Use for linking to websites and external files (e.g., PDFs), as well as to content in other projects (see Inserting Links to External Files—HTML, PDF, Microsoft Office)

Use for linking to other topics within the project when you want specific link text that doesn't necessarily match the destination heading, file name, etc.

Uses MadCap-specific style (MadCap|xref)

Uses standard CSS style (a), which is also used for image hyperlinks and popup links

Can be context-sensitive in print output (e.g., displays "next page," "below," or specific page number)

Not context-sensitive
