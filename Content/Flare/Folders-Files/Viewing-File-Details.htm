Viewing File Details

In the Content Explorer and Project Organizer, you can see the names of content and project files. If you want to see more details for a file (such as the file size, date modified, title, class, and condition tags), you can do so by using the File List window or the split view feature.

You might use these features to apply certain settings to multiple files at once. For example, you can use them to apply stylesheets or condition tags to many topics at the same time.

How to Open the File List Window Pane

Do one of the following:

  • Ribbon Select View > File List.

  • Keyboard Shortcut Press CTRL+SHIFT+J.
