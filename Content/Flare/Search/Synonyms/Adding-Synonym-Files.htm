Adding Synonym Files

The first step to creating synonyms for enhanced search is to add a synonym file to the Flare project.

How to Add a Synonym File

    • Ribbon Select Project > New > Advanced > Synonym File.
    • Right-Click In the Project Organizer, right-click on the Advanced folder and from the context menu select Add Synonym File.

    The Add File dialog opens.

  2. In the File Type field at the top, make sure Synonym File is selected.
  5. In the File Name field, type a new name for the synonym file.
  6. (Optional) See Applying Conditions to Content.
  7. (Optional) See Associating Tags with Files.
  8. Click Add. The synonym file is added to the Advanced folder in the Project Organizer. The Synonyms Editor opens to the right.

After you add a new synonym file, you can create directional synonyms and synonym groups within it. See Creating Synonyms to Enhance Search Results.
