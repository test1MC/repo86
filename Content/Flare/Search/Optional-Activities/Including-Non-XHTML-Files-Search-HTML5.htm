Including Non-XHTML Files in HTML5 Search

 

How to Include Non-XHTML Files in HTML5 Search

  1. Enable HTML5 Server-Based Output  
  2. Generate HTML5 Server-Based Output In Flare, generate a target using the HTML5 output format.
  3. Publish HTML5 Output to Server
  4. (Optional) Add IFilter

  5. Copy and Paste Non-XHTML Files to Appropriate Folder

When testing this search feature on your computer, you need to place the non-XHTML files in the AutoSearch subfolder at this location (as opposed to the output folder that was generated where your Flare project is located).
