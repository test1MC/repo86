Setting Search Result Importance

Search Result Importance

  You must be using Elasticsearch or MadCap Search to use this feature. It is not supported for Google Search.

How to Include or Exclude Importance in Search Results

  1. Open an HTML5 target.
  2. Select the Search tab.
  3. From the Search Engine section, make sure MadCap Search or Elasticsearch is selected.
  4. Expand the Advanced Search Options section.

  5. Select Include Importance.
