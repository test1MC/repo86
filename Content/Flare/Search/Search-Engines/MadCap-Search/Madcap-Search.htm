MadCap Search

MadCap Search is Flare's built-in search engine. When an HTML5 target is created, it is set by default to use this search engine. MadCap Search considers heading styles, index keywords, glossary terms, and more when ranking the search results. You can customize the search engine by setting options such as search result importance, glossary terms in the results, number of search results per page, and abstract character limit.
