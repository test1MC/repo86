Searching a WebHelp System

End users have a variety of options for searching a or system. To learn about the system, see Searching an HTML5 System.

Search Experience

A Standard skin (see Skins) provides the user interface for a published system. When an end user searches a system for information, the results show the topic rank and title. When the user opens a topic, search highlighting reveals the matches.

How to Search

  1. Open the  or system.
  2. In the navigation pane, click Search.

  3. In the entry field, type the search query. Then press ENTER or click Search.

  4. In the search results list, double-click one of the topics in the search list to open it. Search highlighting reveals the matches in the topic.
