You want to keep track of which authors are writing which topics, as well as the status of each topic (e.g., started, in review, completed). As shown below, you can create one file tag set to list all of your authors and another set to list various topic statuses.

Next, you open the Properties dialog for each necessary file (in this case, topics) to associate certain author or status tags with each.
