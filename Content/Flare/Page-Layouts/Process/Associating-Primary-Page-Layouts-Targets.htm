Associating Primary Page Layouts With Targets

 See Page Layouts.

The following steps show how to associate a primary page layout at the target level.

How to Associate a Primary Page Layout With a Target

  1. Open the target. The Target Editor opens.
  2. Select the General tab.
  3. Click in the Primary Page Layout field, and from the drop-down select the page layout that you want to use for all topics in the target.
