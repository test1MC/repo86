Copying and Renaming Page Layouts

How to Copy and Rename a Page Layout

  2. Expand the Resources folder, and double-click the PageLayouts subfolder (unless you have stored these files in a custom folder).
  3. Click on the page layout that you want to copy.
  4. Press CTRL+C on your keyboard.
  5. Press CTRL+V on your keyboard. Another page layout is added, with the same name and the words "Copy of" before it. The settings in that page layout are identical to the settings of the page layout that you copied.
  6. Click once on the new page layout name, so that the main name is highlighted but the file extension is not.
  7. Type a new name for the page layout and press ENTER.

After copying and renaming a page layout, you can open the new page layout to modify its settings as needed. See Editing Pages and Frames.
