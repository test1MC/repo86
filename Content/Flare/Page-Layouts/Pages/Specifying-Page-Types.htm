How to Specify a Page Type

  1. Select the General tab.

  2. Click in the Page Type field and select one of the following:

Continue L/R and Continue FL/FR

In addition to the page types, the following options are available in the Properties dialog for TOC entries. See Specifying Chapter and Page Layout Breaks.

  • (continue L/R) Continues a Left/Right page sequence, using whichever page comes next in the normal flow.
  • (continue FL/FR) Continues a First Left/First Right page sequence, using whichever page comes next in the normal flow.
