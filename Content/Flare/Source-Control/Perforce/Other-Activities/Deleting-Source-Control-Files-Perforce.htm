Deleting Source Control Files—Perforce Helix Core

How to Delete a File

  1. In one of the window panes (e.g., Content Explorer, File List, Project Organizer, Pending Changes window pane), select the relevant file(s).

You can undelete a file from Perforce Helix Core if you need to recover it. See Undeleting Source Control Files—Perforce Helix Core.
