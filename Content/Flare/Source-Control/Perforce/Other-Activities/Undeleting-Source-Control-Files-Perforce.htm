Undeleting Source Control Files—Perforce Helix Core

You can only undelete files from Microsoft Team Foundation Server and Perforce Helix Core. When you delete files from other source control providers, they are removed from the source control repository.

How to Undelete a Source Control File

  1. In one of the window panes (e.g., Content Explorer, Project Organizer, Pending Changes window pane), select the relevant file(s).

    • Ribbon Select Source Control > Undelete.
