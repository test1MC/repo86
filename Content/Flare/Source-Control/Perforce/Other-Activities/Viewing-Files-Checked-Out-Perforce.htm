Viewing Files That are Checked Out—Perforce Helix Core

How to View Files That You and Others Have Checked Out—Pending Changes Window Pane

  1. Select Source Control > Pending Changes.

    • Status

      • See Submitting Source Control Files and Checking Out Source Control Files.
      • See Adding Files to Source Control.
      • Out of Date See Checking Out Source Control Files and Getting the Latest Version of Source Control Files.
      • Locked This indicates that a file has been locked by you or another user. See Locking a File and Unlocking a File.

      See Enabling Source Control Status Checks.

How to View Files That are Checked Out by Others—File List Window Pane

    • Ribbon Select View > File List.
    • Keyboard Shortcut Press CTRL+SHIFT+J.

    • Status

      See Enabling Source Control Status Checks.

How to View Files That You Have Checked Out—Source Control Explorer

  1. Select View > Source Control Explorer.

  2. The Pending Changes pane opens. Files that you have changed appear in the Included Changes or Excluded Changes section (depending on whether you are going to include or exclude them in your next submit; see Submitting Source Control Files).

  3. See Enabling Source Control Status Checks.
