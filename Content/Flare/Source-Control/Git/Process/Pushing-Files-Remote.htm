Pushing Files to a Remote Repository—Git

Pushing Files to MadCap Central

If you are not using MadCap Central (see About MadCap Central and Flare), you can simply use the standard method (described below) for pushing files to a repository. If you are using Central and your project is bound to it (along with a third-party source control tool), you can use the MadCap Central window pane to push your files (see Pushing in a Dual-Bound Model) (for more information, see the online Help).

How to Push Files to a Remote Repository

    • Right-Click If you have the Content Explorer, Project Organizer, Pending Changes window pane, or File List open, right-click on any file and select Source Control > Project > Push.
