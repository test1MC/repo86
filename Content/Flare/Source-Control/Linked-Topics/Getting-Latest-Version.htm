Updating or Getting the Latest Version of Source Control Files

You can get the latest version, update, or synchronize your source control files in the following ways:

  • Committing and Synchronizing (Pull, Push) in a Single-Bound Model
  • Getting the Latest Version of Source Control Files—Perforce Helix Core
  • Getting the Latest Version of Source Control Files—Team Foundation Server
  • Synchronizing Source Control Files—Git
  • Updating Source Control Files—Subversion
