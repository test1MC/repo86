Deleting Source Control Files—Team Foundation Server

You can delete a topic or file that is bound to source control. This also removes the file from Team Foundation Server.

How to Delete a File

  1. In one of the window panes (e.g., Content Explorer, File List, Project Organizer, Pending Changes window pane), select the relevant file(s).
  3. The Delete dialog opens. Select the bound files you want to delete.
  4. Click Delete.
  5. Check in your changes in order for the deletions to be reflected in remote projects. See Checking In Source Control Files—Team Foundation Server.

You can undelete a file from Team Foundation Server if you need to recover it. See Undeleting Source Control Files—Team Foundation Server.
