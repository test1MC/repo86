Key Features

Following are some of the key features of the format.

  • WebHelp Features This format lets you take advantage of features and benefits available with the regular output type.
  • Sitemap/Search Engine Optimization (SEO) See Generating Sitemaps for Search Engine Optimization.
  • Multiple Platforms This format is a good choice if you publish your output to a machine running Windows Server 2003, Windows Server 2008, or Windows 7.
  • Searching of Non-XHTML Content   See Including Non-XHTML Files in Search.
  • Faster Searching
  • Automatic Runtime Merging of Flare Projects See Merging Output at Runtime Using WebHelp Plus.
