Enabling Import/Export Without Word Installed

supports the import and export of Microsoft Word documents without the need to have Word installed locally. For example, a user of Microsoft Office 365 (a subscription-based software solution) would not have Microsoft Word installed locally (via Microsoft Office). However, lets Office 365 users build Word output, or import Word files in the DOCX, DOC, or RTF format, even if they do not have Microsoft Word installed on the same machine.

It is still recommended that you work with Word installed locally, which is a more robust process. However, if you are an Office 365 user without a local installation of Word, you can use the following steps to enable this feature. (This option is disabled by default.)

How to Enable the Import/Export Process Without Word Installed

  1. Select File > Options.
  2. Select the General tab.
  3. Select Import/Export Word Files Without MS Office.
