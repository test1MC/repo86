Adding PDF Files to a Project

You can add PDF files to a Flare project, and subsequently make them part of generated output.

How to Add PDF Files to a Project

  1. The easiest way to do add PDF files to a Flare project is to simply copy and paste them into the Content Explorer. You also might consider using the External Resources feature, which allows you to keep the PDFs in sync over time with the original copies if changes are made to them. For more about External Resources, see the online Help.
  2. Open a table of contents (TOC) file and drag the PDF files to it from the Content Explorer. This is known as "PDF stitching."
  3. Generate output. The PDFs become accessible in online outputs (i.e., you can open them from the TOC). In Flare PDF outputs, they become merged into the final PDF file.

Searching for content in PDFs that you have included in Flare online output is possible only with server-based HTML5 and WebHelp Plus.
