Accessibility General Considerations

When working to make your output accessible, consider the following:

  • Different output formats in support different accessibility features.

  • One way to make content accessible is to provide one output for most users and an entirely separate output for individuals with disabilities. In the locations where you have elements that can present accessibility issues (e.g., images, multimedia), you can provide descriptions of those elements. Then by using condition tags and different targets, you can have customized outputs for each set of users. See Determining the Output Type, About Conditions, Adding Targets, and Editing Target Settings.

  • You can use character-level styles when formatting text. For example, the <strong> tag conveys the idea that you want to emphasize the importance of some words in a paragraph. See About Styles and Stylesheets.
