Troubleshooting User Interface Issues

Following are issues that may occur when using the Flare user interface.

  • Docking indicators do not appear when dragging and dropping in Windows 7

Docking indicators do not appear when dragging and dropping in Windows 7

Cause

This happens when the visual effect settings for Windows 7 are not set for optimal performance.

Solution

Perform the following steps. The steps below may be different depending on your operating system.

  1. Open the Start menu.
  2. Click Control Panel. The Control Panel opens.
  3. In the Control Panel, click Performance Information and Tools.
  4. Click Adjust visual effects. The Performance Options dialog opens.
  5. In the Performance Options dialog, click the Visual Effects tab.
  6. Click Adjust for best appearance.
