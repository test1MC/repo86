Troubleshooting Topic Issues

Following are issues that may occur when working with topics in Flare.

  • "Object reference not set to an instance of object" error occurs when renaming a topic

"Object reference not set to an instance of object" error occurs when renaming a topic

When renaming a topic Flare displays the following error message:

Operation rename failed: "Object reference not set to an instance of an object" when renaming a topic.

If the topic is renamed, all links to that topic are lost.

Cause and Solution

This problem can have one of the following causes.

cause/solution 1

The Flare application data has become corrupted.

Delete the contents of the Flare application data folder. Perform the following steps.

  1. Close the Flare project.
  2. Navigate to C:\Users\[username]\AppData\Roaming\MadCap Software\Flare in the Windows Explorer.
  3. Delete the contents of the Flare folder.
  4. Reopen the Flare project and rename the topic.

cause/solution 2

The window layout needs to be reset.

  1. Select Window > Reset Layout. The window pane layout will reset to default.
  2. Rename the topic.
