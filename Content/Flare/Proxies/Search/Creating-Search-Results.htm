Creating Search Results

How to Create a Page for Search Results

  1. Open a topic that you want to use to hold the search results. Usually you want an empty topic, so remove any excess content. Do not use a template page for holding search results.
  2. Place your cursor in the topic, and select Insert > Proxy > Insert Search Results Proxy.
  3. (Optional) A skin component lets you control the look of generated search results. If you have added multiple skin components to your project, you can use the Skin File field to select the one to associate with this proxy. You can then edit that skin component to change its appearance.
  4. The proxy is added.

Editing a Search Results Proxy

After inserting a Search Results proxy, you can edit it. To do this, open the topic where the proxy is inserted, right-click on it, and select Edit Search Results Proxy.
