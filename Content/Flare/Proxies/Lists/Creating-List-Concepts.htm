Creating a List of Concepts

  

How to Create a List of Concepts

After you use the Concepts proxy to create a list of concepts, you can change the look of that list by using styles. You can modify the look of individual entries in the list, as well as the entire container. See Using Styles to Determine the Look of a List of Concepts.

If you are creating print-based output, add the topic with the Concepts proxy to the outline TOC. See Creating a Table of Contents.
