Viewing Index Keyword Suggestions

You have opened Topic A and inserted "Chair" as an index keyword. The word "Chair" may also occur in the text of Topic B. If you have not yet inserted the word "Chair" as an index keyword in Topic B, it will show up when you view index keyword suggestions.

How to View Index Keyword Suggestions

  1. Open a project.

    • Ribbon Select Analysis > Suggestions > Index Keyword Suggestions.

    • Summary Window Pane Double-click the row that mentions index keyword suggestions.

    The Index Keyword Suggestions window pane opens.

    • Keyword Displays the index keyword text found in the topic that matches an existing keyword.
    • File
    • Title
    • Folder

How to Convert Suggestions to Index Keywords

Do one of the following, depending on whether you want to convert multiple suggestions to index keywords simultaneously, or if you want to convert only one index keyword suggestion at a time.

To Convert One Suggestion at a Time

  1. In the Index Keyword Suggestions window pane, select the row that you want to convert to an index keyword.

  2. The file opens in the application with the suggested content highlighted.

  3. Insert the index keyword. See Adding Index Keywordsthe Flare online Help for more information about inserting index keywords.
