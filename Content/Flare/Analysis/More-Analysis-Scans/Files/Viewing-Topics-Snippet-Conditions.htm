Viewing Files With Snippet Conditions

See Using Snippet Conditions.

How to View Topics With Snippet Conditions

  1. Open a project.

  2. Select the Analysis ribbon. In the Dynamic Reports section select More Reports > Files With Snippet Conditions.

    The Files With Snippet Conditions window pane opens, displaying all files in your project where snippet conditions have been applied.

    • File
    • Location Displays the location of the snippet condition (e.g., topic, snippet block).
    • Folder
    • Expression Displays the condition expression used.
  5. You can open any file in the list to view or modify it.
