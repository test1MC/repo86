How to View Topics Without Concepts

  1. Open a project.
    • Ribbon Select Analysis > More Reports > Topics Without Concepts.
    • Summary Window Pane Double-click the row that mentions topics without concepts.

    The Topics Without Concepts window pane opens, listing all topics that do not have concepts inserted into them.
