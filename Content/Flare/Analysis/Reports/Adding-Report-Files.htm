Adding Report Files

The first step in using analysis reports is to add a report file, which is stored in the Reports folder in the Project Organizer.

How to Add a Report File

    • Ribbon Select Project > New > Add Report File.
    • Right-Click In the Project Organizer, right-click on the Reports folder and from the context menu select Add Report File.

    The Add File dialog opens.

  2. In the File Type field at the top, make sure Report is selected.
  5. In the File Name field, type a new name for the report file.
  6. (Optional) See Associating Tags with Files.
  7. Click Add. The report file is added to the Project Organizer. The Report Editor opens.

After adding a report file, you can provide settings and generate a report. See Generating Reports.
