Setting the Default Paste Behavior

How to Set the Default Paste Behavior

  1. Open a content file (e.g., topic, snippet) and perform one of the actions that causes the paste icon to display.
    • Converting Tables to Text
    • Cutting Table Columns and Rows
    • Merging Tables
    • Pasting Cells Within a Table
    • Pasting Text from Other Programs
    • Inserting Text Hyperlinks
    • Inserting Cross-References
  2. Hover over the icon that appears after the content, click the drop-down arrow, and from the menu select Set Default.
  3. In the XML Editor of the Options dialog, select the paste method that you want to use as the default for that type of action.
