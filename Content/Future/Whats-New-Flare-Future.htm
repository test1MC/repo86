Collapse Tags in XML Editor

You can now collapse tags while working in the XML editor. This allows you to collapse headings, drop-downs, div tags, and snippet blocks.

Span level tags cannot be collapsed at this time in the XML editor. Expanding text blocks cannot be collapsed either.

In the XML editor you see indicators on the left edge to expand or collapse certain blocks. These blocks are expanded by default when you first open a file in the XML editor. There is a - indicator above the respective block in the structure bars area of the editor. You can collapse this block by clicking on the - indicator.

forthcoming imagexxx

You have the option to hide or show the collapse and display indicators in the structure bars. Click on the Show Tags drop-down list in the XML editor to toggle these indicators.

 

You can have as many items collapsed or expanded on block-level tags within a topic. When block-level tags are collapsed with nested content, the nested content will also be hidden under the collapsed block-level tag.

Icons Redesigned in User Interface

The interface has new icons that provide a consistent look and feel to help make Flare easier to use. The prior set of icons had a lot of detail, so the updated icons have been simplified to present a cleaner look in the user interface. These new icons are displayed on ribbons, accordion tabs, and border colors used in Flare. This look is consistent with some of the icons used currently in the Microsoft Office suite.
