Exporting TMX Files

When you have finished translating content in Lingo, you can export a file. After exporting a TMX file, it can be used in other tools such as Trados and memoQ.

How to Export TMX Files

  1. Select Home > Choose TM.
  2. In the Manage Translation Memory dialog, click Export.
  3. Next to the TMX File field, click .
  4. In the dialog that opens, navigate to the folder where you want to export the file. Also in the File name field, enter a name for the new TMX file. Then click Save.
  5. In the Languages to export field, all of the languages involved in your translation memory database are listed, showing both the source and target languages. Select the languages that you want to include in the exported TMX file.

  6. In the Source language field, select the language(s) that you want to be available as the source in the exported TMX file. You can select an individual language or you can select "Any," which allows tools to use translations in any direction in the TMX file.

  7. Click Start Export. A message lets you know the results of the export.
