Learning and Development

Flare has expanded its authoring capabilities into the Learning and Development (L&D) space. You can now create, manage, and build interactive eLearning courses while leveraging the many powerful authoring features that already exist in Flare. For all of the details on this new feature, see Learning and Development.For all of the details on this new feature, see the online Help.

eLearning Capabilities

There are many eLearning authoring tools that learning teams can choose from, so why use Flare? The following outlines key benefits and capabilities. See Flare's eLearning Benefits and Capabilities.

eLearning Features

Key eLearning features allow you to:

  • Create interactive courses with multiple choice or multiple response questions in the XML Editor.

  • Create a quiz (i.e., test) using HTML5 output.

    • Assign a passing score.

    • Randomize answers.

    • Customize pass and fail test results pages.

    • Limit the number of test attempts.

  • Track courses with an external LMS.

    • Generate LMS zip packages to standards xAPI (Tin Can), SCORM 2004, SCORM 1.2.

    • Track to the test score or to course completion.

  • Include questions with an option to mark correct answers in PDF output.

Updates to the Interface

Look for the following updates and additions to the Flare interface:

  • New eLearning templates in the Start New Project Wizard to quickly create linear test workflows. See Creating a Project.

  • New eLearning ribbon with controls to create multiple choice or multiple response questions, tests, feedback, and more.

  • New Question Properties window pane to edit questions, generate IDs, and assign correct answers.

  • XML Editor updates with MadCap-specific tags and corresponding styles to support eLearning authoring.

  • New eLearning Toolbar proxy that acts as a placeholder for navigation controls.

  • New Test Results proxy that acts as a placeholder for a custom results page.

  • New eLearning Toolbar skin to customize the appearance of navigation for course output.

  • New Test Results skin to customize page showing quiz results.

  • A test node in the table of contents for enabling gradable test options, such as assigning a passing score, or linking to a pass or fail page.

  • New eLearning tab in HTML5 target with L&D options.

Process

The following outlines the main parts of a course that make it work in Flare. They are important to know about if you want to create a course without using the eLearning template. For example, you might need to create eLearning content from the ground up if you have an existing project, and you want to add course components to it. See the online Help for detailed content for each step of the process.

At a minimum, all you have to do is:

1. Create a project based on one of the eLearning templates.

2. Replace the eLearning content with your own.

3. Build the target.
