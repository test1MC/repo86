    The levels always begin with the MiniTOC1 style. For example, if the Mini-TOC proxy was placed in a topic under a heading using the h2 style, the first level in the generated mini-TOC still uses the style MiniTOC1 (not MiniTOC3).

    For mini-TOCs in print output, the page numbers in the generated list inherit their style settings from the list items before them. However, if you want the page numbers to look differently, you can modify the MadCap|xref.TOCPageNumber style.
