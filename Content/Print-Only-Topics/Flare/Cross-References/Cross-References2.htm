Cross-References

 

You can use the built-in analysis tool and reports to find broken links. See Viewing Broken Links.

You can generate a report to see places in your project where you might want to create a cross-reference. Essentially, Flare finds files where a hyperlink to a topic exists. Because cross-references are generally preferred over hyperlinks (due to the fact that they are dynamic), it suggests that you create a cross-reference instead. See Reports and Generating Reports.
