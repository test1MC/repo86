Restarting Footnote Numbering

 

How to Restart Footnote Numbering

  1. If Using Simplified View

    5. Select the Footnote tab.

    6. From the Number field, select one of the following.

    If Using Advanced View

    6.  

    7. To the right of mc-footnote-number, click , and select one of the following:

      • chapter This restarts the footnote numbering at each chapter of the output.
      • endnote This is designed to maintain the numbering with a group of endnotes inserted at a specific location in the output, such as at the end of the book. It works in conjunction with an Endnotes proxy, which you create separately. The endnote comments and their numbers are displayed wherever you insert the Endnotes proxy.
      • page This restarts the footnote numbering on each page of the output.
      • section This restarts the footnote numbering at each section of the output.
