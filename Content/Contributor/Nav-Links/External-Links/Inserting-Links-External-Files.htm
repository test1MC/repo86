Inserting Links to External Files—HTML, PDF, Microsoft Office

An external link lets you search for a file (e.g., HTM, HTML, XML, PDF, Microsoft Office files) outside your document.

How to Insert a Link to an External File

  1. Open the where you want to insert the link.
  2. Use the normal steps for inserting the type of navigation link that you want to create:
    • Inserting Text Hyperlinks
    • Inserting Image Hyperlinks
  3. From the Link to drop-down field, select External File.

  4. In the field to the right, next to the External File button, enter the path for the link.

  5. (Optional) You can click to display and select any bookmarks within the destination file, or if you have selected a PDF file.
  6. In the link dialog, click OK.
