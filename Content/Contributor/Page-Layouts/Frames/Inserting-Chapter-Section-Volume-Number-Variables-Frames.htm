Inserting Chapter, Section, and Volume Number Variables into Frames

How to Insert a Chapter, Section, or Volume Number Variable

    • Ribbon Select Insert > Variable.
    • Local Toolbar Click .
    • Keyboard Shortcut On your keyboard press CTRL+SHIFT+V.

    The Variables dialog opens, with the variable set(s) on the left and the variables associated with the selected set on the right.

  3. Select the System variable set.

  4. Select the appropriate variable: ChapterNumber, SectionNumber, or VolumeNumber.

  5. The variable is added to the Frame Contents window pane.
  6. The variable can now be seen in the frame.

In addition to completing the these steps, make sure you also perform the necessary tasks for creating the autonumbers in your topics and specifying where new chapters, sections, or volumes should begin.

  • To generate chapter numbers, you need to create an autonumber format that includes the {chapnum} command. Then you need to specify chapter breaks in the outline TOC.
  • To generate section numbers, you need to create an autonumber format that includes the {secnum} command. Then specify section breaks in the outline TOC.
  • To generate volume numbers, you need to create an autonumber format that includes the {volnum} command. Second, you need to specify chapter breaks in the outline TOC. Third, you need to specify the autonumber flow for each volume, resetting the volume number to a specific number.
