Opening Page Layouts

You can open page layouts that you have created or that exist in a folder on your computer. However, you cannot open the page layouts that are associated with contribution templates or topics sent to you by Flare authors.

How to Open a Page Layout From a Location on Your Computer

    • Select File > Open.

    • Press CTRL+O on your keyboard.
  2. Click in the Files of type field and select MadCap Page Layout Files (*.flpgl, *.mcpgl).
  3. Navigate to the page layout file that you want to open and double-click it.

    4. The page layout opens in the Page Layout Editor.

How to Open a Page Layout That is Part of a New Document

The following steps pertain to new documents that you create without using a contribution template from a Flare author.

  1. Open the document that you created.

  2. Right-click on the margin of the page (i.e., not on a frame). Then in the menu, select Edit Page Layout Document.

    The page layout opens in the Page Layout Editor.

After opening a page layout, you can add pages to the layout. You can also add frames to the pages and edit them. See Adding Pages, Adding Page Layout Frames, and Editing Pages and Frames.
