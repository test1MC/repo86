File Tagging

An author creates tags in Flare in order to keep track of the status of files (e.g., In Progress, Sent for Review, Completed). The author then sends a topic to you for review, with the request that you mark the file as completed after you are finished.

In the Review Packages window pane, you right-click the file and select Properties. This opens the Properties dialog for that file. In that dialog you select the File Tags tab and choose the appropriate option.

You can then return the topic to the author, who adds it to the Flare project.

These tags are intended to be assigned to entire files, not to paragraphs or other content within a topic.
