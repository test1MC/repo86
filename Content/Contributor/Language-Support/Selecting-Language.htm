Selecting a Language

You can use the following steps to select a language for all documents or for a specific document.

How to Select a Language for All Documents

  1. Select File > Options.
  2. Select the Language tab.
  3. From the list, select a language (those in bold font are supported in spell check).

How to Select a Language for a Specific Document

  1. Open a document.
    • Ribbon Select File > Properties.
    • Keyboard Shortcut Press CTRL+SHIFT+P on the keyboard.

    The Properties dialog opens.

  3. Select the Language tab.
  4. From the list, select a language (those in bold font are supported in spell check).
