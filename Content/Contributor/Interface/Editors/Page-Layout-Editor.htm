What You Can Do in the Page Layout Editor

Following are the primary tasks that you can perform in the Page Layout Editor.

  • Bleed, Crop Marks, Registration Marks (Show) See Showing the Bleed, Crop, and Registration Marks in Page Layouts.
  • Frames (Add) See Adding Page Layout Frames.
  • Frames (Remove) See Removing Page Layout Frames.
  • Margins (Show) See Showing Margins in Page Layouts.
  • Padding (Show) See Showing Padding in Page Layouts.
  • Page Numbers/Variables (Add to Frame) See Select Frame Text Dialog.
  • Page Properties (Set) See Page Properties Dialog.
  • Pages (Add) See Adding Pages.
  • Pages and Frames (Edit) This might consist of various tasks, such as inserting page numbers or text, adding borders to frames, resizing frames, and more. See Editing Pages and Frames.
  • Text (Add to Frame) See Adding Text and Content to Frames.

Local Toolbar

See Page Properties Dialog.

  • Copy Page Copies the active page. See Copying Pages in Page Layouts.
  • Paste Page Pastes the page, thus creating a new one. See Copying Pages in Page Layouts.
  • Duplicate Page Copies and pastes the page, thus creating a new one. See Duplicating Pages in Page Layouts.
  • Remove Page Removes the selected page from the layout. See Removing Pages from Page Layouts.
  • Add Page Opens a menu that lets you add one of the following types of pages. See Inserting Pages in Page Layouts.
  • Mirror Page See Mirroring Pages in Page Layouts.
  • Page Properties Opens the properties dialog for the page.
