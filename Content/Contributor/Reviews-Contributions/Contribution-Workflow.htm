Contribution Workflow

Introduction

Following is the basic workflow if you are creating original content to be contributed to a MadCap Flare project.

  1. (Optional) Save Contribution Template See Saving Contribution Templates.

  2. Create You can create new documents from the interface. See Creating New Documents.
  3. (Optional) Insert Comments You can insert annotations (comments) that are associated with specific areas of text in the document. These annotations do not become part of the content in the document, but rather are intended for internal use between you and others. See Inserting Annotations.
